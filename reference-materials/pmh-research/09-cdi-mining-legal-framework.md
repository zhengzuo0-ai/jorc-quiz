## 执行摘要（Executive Summary）

科特迪瓦（Côte d'Ivoire, CDI）已跃升为西非乃至整个非洲大陆最具活力的黄金矿业投资目的地之一。国际货币基金组织（IMF）2025年7月报告指出，CDI 经济增速达6.0%，其中矿业部门扩张13.9%，为主要增长引擎之一（来源：[IMF 2025 Article IV Consultation](https://www.imf.org)）。在2025年5月于阿比让举办的矿业大会上，多位行业高管将 CDI 称为"领先的黄金矿业管辖区"（来源：[Ecofin Agency](https://www.ecofinagency.com/news/0509-48458-at-mining-conference-executives-call-cote-d-ivoire-the-leading-gold-mining-jurisdiction)）。

PMH 已在 CDI 签署10个小型采矿许可证和1个大型许可证，目标覆盖面积达3,000km²。然而，根据现行《矿业法》（Code Minier, Loi N° 2014-138）规定，单一主体（single entity）持有的勘探许可面积存在上限约束，因此实现3,000km² 目标需要通过多个特殊目的载体（SPV, Special Purpose Vehicle）进行架构设计。同时，CDI 政府于2024年通过《受益所有人登记法》（Beneficial Ownership Register Law）并颁布实施细则（Décret 2024-583），对 UBO（Ultimate Beneficial Owner，最终受益所有人）的透明度要求日益提高。

本报告旨在为 PMH 提供：
1. CDI 矿业法规框架的全面解读（含2024-2025年最新修正）；
2. 关键法律问题（面积上限、NSR、政府Free Carry、本地化要求）的深度分析；
3. 主要外资矿业公司在 CDI 的架构与经验教训；
4. 完整的税收体系梳理；
5. 社区发展协议（CDA）模板与最佳实践；
6. 针对 PMH 3,000km² 目标的具体合规架构方案与入驻法律清单。

## 9.1 科特迪瓦矿业法规框架

### 9.1.1 2014年《矿业法》逐章解读（Code Minier, Loi N° 2014-138 du 24 mars 2014）

2014年3月24日，科特迪瓦国民议会通过了第2014-138号法律，即新《矿业法》（Code Minier），取代了此前的1995年矿业法。该法律与其实施细则（Décret d'Application N° 2014-397 du 25 juin 2014）共同构成了 CDI 矿业活动的基本法律框架（来源：[Droit-Afrique](https://www.droit-afrique.com/uploads/RCI-Code-minier-2014.pdf)；[Mayer Brown Analysis](https://www.mayerbrown.com/-/media/files/perspectives-events/publications/2015/07/cote-divoire-new-mining-code/files/new-mining-code/fileattachment/new-mining-code.pdf)）。

**第一编：一般规定（Dispositions Générales）**

- 确立国家对矿产资源的主权所有权（Article 3）；
- 明确矿产资源属于国家财产（domaine public de l'État），任何自然人或法人不得对地下矿产资源主张所有权；
- 矿业活动需取得国家授权（autorisation）或许可（permis）方可开展。

**第二编：矿业权制度（Régime des Titres Miniers）**

矿业权分为两大类别：

**A. 授权制度（Régime d'Autorisation）** — 适用于半工业化开采和手工开采：
- 半工业化开采授权（Autorisation d'Exploitation Minière Semi-Industrielle）
- 手工开采授权（Autorisation d'Exploitation Minière Artisanale）

**B. 许可制度（Régime de Permis）** — 适用于正式勘探和工业化开采：
- 勘探许可（Permis de Recherche / Exploration Permit）
- 采矿许可（Permis d'Exploitation / Mining/Exploitation Permit）

**第三编：环境保护与社区发展（Protection de l'Environnement et Développement Communautaire）**

- 环境影响评估（ESIA, Étude d'Impact Environnemental et Social）为强制要求；
- 矿业公司须制定环境管理计划（Plan de Gestion Environnementale）；
- 社区发展计划（Plan de Développement Communautaire）与社区发展基金（Fonds de Développement Communautaire）为法定义务。

**第四编：财税制度（Régime Fiscal et Douanier）**

- 规定矿业特许权使用费（Redevance Minière / Mining Royalty）费率；
- 稳定条款（Clause de Stabilité）机制；
- 进口关税优惠。

**第五编：罚则与争端解决（Pénalités et Règlement des Différends）**

- 违反矿业法的行政处罚与刑事处罚；
- 争端解决优先采用协商机制，必要时可提交国际仲裁。

### 9.1.2 许可证类型详解

#### （一）勘察许可（Autorisation de Prospection / Reconnaissance Licence）

| 项目 | 内容 |
|------|------|
| 法文名称 | Autorisation de Prospection |
| 适用对象 | 任何自然人或法人 |
| 面积限制 | 无明确面积上限，但须提交工作计划 |
| 有效期 | 2年，可续期1次 |
| 权利范围 | 非排他性地表勘察，不含钻探 |
| 审批机关 | 矿业部长（Ministre en charge des Mines） |

#### （二）勘探许可（Permis de Recherche / Exploration Permit）

| 项目 | 内容 |
|------|------|
| 法文名称 | Permis de Recherche |
| 适用对象 | 法人实体（公司），须满足技术和财务资质要求 |
| **面积限制** | **单个许可证不超过 400km²**（旧法为1,000km²） |
| **有效期** | **初始4年，可续期2次，每次3年（共计最长10年）** |
| 续期条件 | 每次续期须放弃原面积的 1/4（除非证明全部面积均在实际使用中，可申请保留全部面积，但须缴纳额外费用） |
| 排他性 | 排他性勘探权（droit exclusif de recherche） |
| 最低投资要求 | 前4年最低勘探预算为每平方米 1,600 FCFA（约 2.44 欧元/m²）|
| 开工要求 | 许可颁发后6个月内须开始勘探工作 |
| 技术要求 | 申请人须在过去10年内承担过至少2个勘探项目，且须聘用具有不少于7年勘探经验的技术经理 |

（来源：[ICLG Mining Laws & Regulations - Ivory Coast 2025](https://iclg.com/practice-areas/mining-laws-and-regulations/ivory-coast)；[Lexology - Overview and Outlook](https://www.lexology.com/library/detail.aspx?g=488acfcd-5c37-4bf1-8836-93644cfa7a22)；[Mayer Brown](https://www.mayerbrown.com/-/media/files/perspectives-events/publications/2015/07/cote-divoire-new-mining-code/files/new-mining-code/fileattachment/new-mining-code.pdf)）

#### （三）采矿许可（Permis d'Exploitation / Mining Permit）

| 项目 | 内容 |
|------|------|
| 法文名称 | Permis d'Exploitation |
| 适用对象 | 持有有效勘探许可并完成可行性研究的法人实体 |
| 面积限制 | 以勘探许可范围内证实的矿区为限 |
| **有效期** | **按矿山寿命确定，最长20年** |
| **续期** | **可多次续期，每次最长10年** |
| 排他性 | 排他性开采权，含运输、加工、精炼和贸易权 |
| 矿业协议 | 须在获得采矿许可后60个工作日内与国家签署矿业协议（Convention Minière），初始期限12年，可续期（每次最长10年） |

#### （四）半工业化开采授权（Autorisation d'Exploitation Minière Semi-Industrielle）

| 项目 | 内容 |
|------|------|
| 法文名称 | Autorisation d'Exploitation Minière Semi-Industrielle |
| 适用对象 | 科特迪瓦国籍自然人，或由科特迪瓦国民持多数股权的中小企业/合作社 |
| 面积限制 | 较小区域（通常不超过数十km²） |
| 有效期 | 4年，可续期 |
| 特殊要求 | 外资公司不能直接持有，须通过本地合资结构 |

（来源：[Loidici.biz - Code Minier](https://loidici.biz/2020/02/17/chapitre-premier-autorisation-dexploitation-miniere-semi-industrielle/lois-article-par-article/codes/le-code-minier/17610/naty/)）

### 9.1.3 2024-2025年最新修正与政策动态

**（一）2025年《财政法》（Loi de Finances 2025）对矿业税制的修改**

2024年底通过的2025年《财政法》对矿业税制进行了重大调整：

- **黄金矿业特许权使用费由原来的3%-6%浮动费率统一调整为8%的固定费率**。此前费率依矿业协议条款而异，新政策取消了协议谈判空间，实行"一刀切"（来源：[TRT Afrika](https://www.trtafrika.com/english/article/c8ab094aa8fe)；[Crux Investor - Perseus Mining](https://www.cruxinvestor.com/posts/perseus-mining-absorbs-ivory-coast-royalty-reset-highlights-tier-one-operator-advantages)）。
- 这一变化引发了行业争议，多家矿业公司对新费率的合法性提出质疑，但最终于2025年开始执行，且追溯至2025年1月生效。

**（二）受益所有人登记制度（Registre des Bénéficiaires Effectifs）**

- 2024年4月，CDI 政府通过法律，建立受益所有人登记簿制度；
- 实施细则（Décret 2024-583）对公开访问范围进行了限制：虽然法律第15条规定登记簿对公众开放，但实施细则第8条将公众可访问信息限制为报告实体的一般信息，不包括受益所有人的具体数据（来源：[EITI Côte d'Ivoire](https://eiti.org/countries/cote-divoire)）。

**（三）手工采矿规范化倡议**

- 2025年7月11日，CDI 与世界银行和世界黄金协会（World Gold Council）签署合作协议，旨在将其庞大的非正式小规模采金业纳入规范化管理（来源：[World Bank Press Release](https://www.worldbank.org/en/news/press-release/2025/07/11/cote-ivoire-moves-to-formalize-its-vast-informal-gold-mining-sector)）。

**（四）数字化矿业地籍系统**

- CDI 正在推行电子矿业地籍平台（e-cadastre minier），以提高透明度并满足 EITI（Extractive Industries Transparency Initiative，采掘业透明度倡议）合规要求（来源：[Ecofin Agency](https://www.ecofinagency.com/news-digital/2409-48941-cote-d-ivoire-embraces-digital-mining-cadastre-trend)）。

**（五）企业所得税免税期的取消**

值得注意的是，《矿业法》第169条(e)款原规定采矿许可持有人在商业生产开始后前5年享有企业所得税和最低固定税的全面豁免，但该优惠已被2018年第2018-144号法令（Ordonnance No. 2018-144）废除（来源：[Lexology](https://www.lexology.com/library/detail.aspx?g=488acfcd-5c37-4bf1-8836-93644cfa7a22)）。

## 9.2 关键法律问题深度研究

### 9.2.1 面积上限问题：法律原文、执法实践与多SPV架构

**（一）法律规定**

根据《矿业法》及其实施细则，单个勘探许可（Permis de Recherche）的面积上限为 **400km²**。该上限相较1995年旧法的1,000km² 已大幅缩减。

对于采矿许可（Permis d'Exploitation），面积以可行性研究确定的矿区范围为限，无统一的绝对面积上限，但实际操作中受制于勘探许可的覆盖范围。

**（二）单一主体累计面积约束**

《矿业法》本身未明确规定单一法人实体可持有的勘探许可累计面积上限。然而，在实践中，矿业行政管理部门（Direction Générale des Mines et de la Géologie, DGMG）对单一实体或关联实体集团持有的许可总面积存在行政审核门槛。根据行业惯例和部分法律分析报告，**单一主体或其关联方的累计勘探面积上限通常被理解为约2,000km²**，超过此限度可能触发额外审查。

> **PMH 关键风险点：** PMH 目标3,000km² 超过了行业认知中的单一UBO 2,000km² 门槛。这意味着：
> - 需要至少 **2个独立SPV**（特殊目的载体）分别持有许可证；
> - 每个SPV须具备独立的法律人格、独立的董事会构成和独立的运营团队；
> - SPV之间的关联关系须经审慎设计，以避免被认定为"规避"（circumvention）。

**（三）同一UBO多SPV是否被视为规避**

这是 PMH 面临的核心法律风险。分析如下：

1. **法律文本层面：** 2014年《矿业法》未明确禁止同一UBO通过多个法人实体持有许可。法律条文使用的主语是"持有人"（titulaire），通常指被登记为许可持有人的法人实体本身。

2. **2024年受益所有人登记制度的影响：** 2024年通过的《受益所有人登记法》要求所有企业披露UBO信息。虽然实施细则限制了公众对UBO数据的访问，但矿业主管部门有权调阅完整的UBO信息。这意味着通过多SPV持有超限面积的安排可能被主管部门识别。

3. **执法实践：** 目前 CDI 尚未出现因多SPV结构而被认定规避面积上限并撤销许可的公开案例。主要国际矿业公司（如 Endeavour Mining）通过不同子公司分别持有不同矿区的做法已被普遍接受。关键在于每个SPV须具有**实质性的商业目的**（substance）和**独立的运营逻辑**。

4. **风险缓解建议：**
   - 每个SPV应具有不同的少数股东（minority shareholder）；
   - SPV之间的管理层应有差异化安排；
   - 每个SPV应有独立的勘探计划和独立的银行账户；
   - 建议在申请许可前通过本地顶级律所（如 GENI & KEBE、LPA-CGR、Bilé-Aka 等）获得正式法律意见书（legal opinion）。

### 9.2.2 NSR条款（Net Smelter Royalty）的法律含义

**（一）NSR 定义**

Net Smelter Royalty（NSR，冶炼厂净回报特许权使用费）是指矿山运营方须向NSR权利持有方支付的、基于矿产品销售净收入（扣除运输和冶炼加工费用后）的一定比例的费用。典型的NSR费率在1%-3%之间，PMH 相关许可涉及的NSR约为1%-2%（来源：[Wikipedia - Net Smelter Return](https://en.wikipedia.org/wiki/Net_smelter_return)；[Oracle Holding](https://oracleholding.com/mining-nsr/)）。

**（二）CDI 法律框架下的NSR**

CDI《矿业法》本身并不直接规定NSR，NSR通常是矿业权转让交易中的商业条款，由合同约定。在CDI法律框架下：

- NSR属于合同约定的私法义务（obligation contractuelle de droit privé）；
- 不受《矿业法》中政府矿业特许权使用费（redevance minière）制度的直接规制；
- 但须遵守CDI《合同法》（OHADA统一法体系下的合同法）的一般规定。

**（三）NSR持有方的权利范围分析**

| 权利类型 | 通常约定 | PMH应关注的要点 |
|----------|----------|-----------------|
| 信息权（Information Rights） | 有权获取产量和销售数据 | 须确保信息披露范围可控 |
| 审计权（Audit Rights） | 有权对NSR计算进行审计 | 审计频率和费用承担须明确 |
| **否决权（Veto Rights）** | **通常无**，但高比例NSR合同中偶有约定 | 须逐一审查PMH现有NSR合同 |
| **优先购买权（Right of First Refusal, ROFR）** | **视合同而定**，部分NSR合同包含ROFR条款 | 如存在，可能限制许可转让灵活性 |
| **参与权（Participation Rights）** | **罕见** | 须确认PMH合同中是否存在 |
| **回购权（Buyback Option）** | 部分合同允许运营方以约定价格回购NSR | 如有，建议在合适时机行使 |

**（四）PMH NSR风险缓解策略**

1. 立即由Merlanih对全部11个许可证相关的NSR合同进行全面审查；
2. 识别具有ROFR或否决权条款的高风险合同；
3. 评估回购可行性和成本；
4. 对于新获取的许可证，在谈判中坚持：NSR上限不超过1.5%、不含ROFR、不含否决权。

### 9.2.3 政府Free Carry（Government Free Carried Interest）

**（一）法律规定**

《矿业法》第7条（Article 7）明确规定：

> *"L'attribution par l'État du permis d'exploitation donne droit, en contrepartie de la richesse distribuée et de l'appauvrissement du sous-sol, à l'attribution à l'État d'actions équivalentes à 10% du capital social de la société pour la durée de la mine."*

翻译："国家授予采矿许可的同时，作为对已分配的财富和地下资源贫化的对价，国家有权获得相当于矿业公司股本10%的股份，期限为矿山存续期间。"

**（二）关键条款分析**

| 条款 | 内容 |
|------|------|
| 持股比例 | **10%** 的公司股本 |
| 出资要求 | **无** — "Aucune contribution financière ne peut être exigée de l'État au titre de ses actions, même en cas d'augmentation du capital social"（即使增资时亦不得要求国家出资） |
| 稀释保护 | **非稀释性**（non-dilutable）— 国家持股比例在增资时不被稀释 |
| 额外参与权 | 国家有权以市场条件另外购买最多 **15%** 的股权（即累计最高可达 **25%**） |
| 股东权利 | 国家作为股东享有分红权、信息权和投票权 |

（来源：[Fasken - CDI New Mining Code](https://www.fasken.com/en/knowledge/2014/03/miningbulletin-20140318)；[Lexology - Free Carried Interests in Francophone Africa](https://www.lexology.com/library/detail.aspx?g=7e65240f-b85e-4162-9af8-f8ec70ba0136)；[Crux Investor](https://www.cruxinvestor.com/posts/mining-in-cote-divoire-africas-emerging-mining-powerhouse)）

**（三）PMH架构影响**

- 每个持有采矿许可的SPV均须向CDI政府无偿出让10%股权；
- 若PMH通过2个SPV运营，则2个SPV各自须出让10%给政府，但由于SPV独立，政府在每个SPV中的绝对话语权相对有限；
- 须注意政府行使额外15%购买权的可能性 — 建议在矿业协议谈判中明确政府额外购买权的行使期限和条件。

### 9.2.4 本地化要求（Local Content Requirements）

《矿业法》建立了全面的本地化要求体系（来源：[Lexology](https://www.lexology.com/library/detail.aspx?g=488acfcd-5c37-4bf1-8836-93644cfa7a22)；[ICLG](https://iclg.com/practice-areas/mining-laws-and-regulations/ivory-coast)）：

**（一）雇员本地化**

- 许可持有人须优先雇用科特迪瓦国民（personnel ivoirien）；
- 须制定并资助培训计划，以提高科特迪瓦国民的就业能力；
- 行业最佳实践显示，主要矿业公司本地员工比例通常达到90%以上（Endeavour Mining 在 Ity 矿的本地员工比例达93%）。

**（二）本地采购**

- 须优先使用具有能力的科特迪瓦本地企业提供商品和服务；
- 须建立本地企业培训和能力建设计划；
- 须向矿业行政部门定期报告本地采购比例。

**（三）社区发展基金（Fonds de Développement Communautaire）**

- **法定缴纳比例：采矿许可持有人须将年营业额的 0.5% 注入社区发展基金**；
- 基金由地方矿业发展委员会（Comité de Développement Local Minier, CDLM）管理；
- 基金用于资助受影响社区的社会经济发展项目；
- 该支出可从企业所得税税基中扣除。

**（四）技术转让与能力建设**

- 须为矿业行政管理机构的能力建设作出贡献；
- 须提交技术转让计划。

## 9.4 税收体系完整梳理

### 9.4.1 税种与税率总览

| 税种 | 法文名称 | 费率/税率 | 备注 |
|------|----------|-----------|------|
| **企业所得税** | Impôt sur les Bénéfices Industriels et Commerciaux (BIC) | **25%** | 适用于工业和商业利润，设有最低税额 |
| **矿业特许权使用费（黄金）** | Redevance Minière (Or) | **8%**（2025年起） | 2025年《财政法》修正，此前为3%-6%浮动费率 |
| **矿业特许权使用费（其他矿种）** | Redevance Minière (Autres) | 视矿种而定，通常2.5%-5% | 依据矿种和市场价格 |
| **增值税** | Taxe sur la Valeur Ajoutée (TVA) | 18% | 勘探阶段进口设备可豁免 |
| **社区发展基金** | Fonds de Développement Communautaire | 年营业额的 **0.5%** | 可从企业所得税税基中扣除 |
| **红利预扣税** | Retenue à la source sur dividendes | 10%-15% | 视双边税收协定而定 |
| **土地税/水资源税** | Taxes foncières / Taxes sur l'utilisation de l'eau | 减免适用 | 矿业法规定特殊减免 |

（来源：[PwC Tax Summary - Ivory Coast](https://taxsummaries.pwc.com/ivory-coast/corporate/taxes-on-corporate-income)；[EY Mining & Metals Tax Guide](https://s3.amazonaws.com/rgi-documents/a3a7e2ead5afcbd4626f6a0edeb76e32381b8945.pdf)；[TRT Afrika](https://www.trtafrika.com/english/article/c8ab094aa8fe)）

### 9.4.2 矿业特许权使用费详解

**2025年改革前（旧制度）：**
- 黄金矿业特许权使用费为 **3%-6%** 的浮动费率；
- 具体费率在各个矿业协议（Convention Minière）中协商确定；
- 存在滑动比例机制（sliding scale），费率与黄金现货价格挂钩。

**2025年改革后（新制度）：**
- 黄金矿业特许权使用费统一为 **固定8%**；
- 取消了此前的协商空间和浮动机制；
- 追溯自2025年1月1日起生效；
- 此次改革被业界解读为CDI政府希望在黄金价格持续走高的背景下获取更大的财政收入份额。

**西非区域对比：**

| 国家 | 黄金矿业使用费费率 | 备注 |
|------|---------------------|------|
| 科特迪瓦（CDI） | 8%（固定） | 2025年起 |
| 加纳（Ghana） | 5% | 固定费率 |
| 布基纳法索（Burkina Faso） | 3%-5%（浮动） | 与金价挂钩 |
| 马里（Mali） | 3%-6% | 视协议条款 |
| 塞内加尔（Senegal） | 5% | 固定费率 |

### 9.4.3 进口关税豁免条件

根据《矿业法》的财税制度规定（来源：[PwC - Tax Credits and Incentives](https://taxsummaries.pwc.com/ivory-coast/corporate/tax-credits-and-incentives)；[Mayer Brown](https://www.mayerbrown.com/en/insights/resource-centers/africa-mining-know-how/ivory-coast)）：

**勘探阶段：**
- 用于勘探活动的材料、机械和设备的进口可**免征全部进口税费和增值税**。

**生产阶段：**
- 矿业分包商进口液态或气态燃料、润滑剂以及用于矿物处理的化学或有机产品，在**整个矿山运营期间**免征关税（含增值税）；
- 大型设备和资本货物的进口可享受关税减免，但须在矿业协议中具体约定。

### 9.4.4 利润汇回限制

CDI 作为西非经济货币联盟（UEMOA / WAEMU）成员国：

- **货币：** 使用西非法郎（XOF / FCFA），与欧元挂钩，固定汇率1 EUR = 655.957 FCFA，对欧元的无限兑换能力由法国财政部担保；
- **矿产品销售收入汇回义务：** 矿产品销售所得收入须在付款到期日后1个月内汇回CDI境内，付款日须在装运日后120天内；
- **资本和利润转移：** 对以可兑换外汇进行的投资所产生的资本和收入的转移或汇回不设限制。政府在投资建立并有文件记录后，通常会批准股息汇出和/或资本汇回。

（来源：[US State Department - 2025 Investment Climate Statement](https://www.state.gov/reports/2025-investment-climate-statements/cote-divoire)）

### 9.4.5 稳定条款（Stability Clause / Clause de Stabilité）的实际保护力度

**（一）法律机制**

矿业协议（Convention Minière）是税收和关税稳定的载体。《矿业法》和矿业协议共同保障开采阶段的税收和关税制度稳定性。

- 矿业协议初始期限 **12年**，可续期（每次最长10年）；
- 协议内容须符合《矿业法》中的税收和关税规定；
- 签署时间：获得采矿许可后 **60个工作日** 内。

**（二）实际保护力度评估**

> **关键案例 — 2025年黄金矿业使用费调整事件：**

2025年《财政法》将黄金矿业使用费从3%-6%调整为固定8%，且追溯适用。这一做法引发了关于稳定条款保护力度的严重质疑：

- 多家矿业公司认为此举违反了矿业协议中的稳定条款，因为协议中约定的费率低于新法定的8%；
- 然而，CDI政府立场是《财政法》属于主权立法行为，可超越矿业协议中的约定；
- 矿业公司最终接受了新费率并开始按8%缴纳，这表明**稳定条款的实际保护力度弱于预期**。

**（三）对PMH的启示**

- 不能过度依赖稳定条款作为财税风险的唯一保障；
- 财务模型须纳入政府单方面调整税率的情景分析；
- 建议在矿业协议中争取**国际仲裁条款**（如ICC或ICSID仲裁），作为稳定条款被突破时的最终救济途径；
- 关注CDI是否为投资者所在国签署了**双边投资保护协定**（BIT, Bilateral Investment Treaty）。

## 9.5 社区发展协议（CDA）模板与最佳实践

### 9.5.1 法律要求的最低内容

根据《矿业法》及其实施细则，采矿许可持有人须与受影响社区签订社区发展协议（Convention de Développement Communautaire / Community Development Agreement, CDA）。法定最低内容包括（来源：[World Bank - Mining CDA Source Book](https://documents1.worldbank.org/curated/en/522211468329663554/pdf/712990WP0minin00Box370065B00PUBLIC0.pdf)；[Open Government Partnership - CDI](https://www.opengovpartnership.org/members/Cote%20dIvoire/commitments/CI0002/)）：

1. **社区发展计划**（Programme de Développement Communautaire）— 与受影响社区和行政当局共同制定；
2. **社区发展基金**（Fonds de Développement Communautaire）— 年营业额的0.5%注入基金；
3. **环境保护措施**（Mesures de Protection Environnementale）；
4. **土地征用补偿方案**（Modalités d'Indemnisation pour l'Acquisition de Terres）；
5. **就业和培训安排**（Dispositions en Matière d'Emploi et de Formation）；
6. **争端解决机制**（Mécanisme de Règlement des Différends）。

### 9.5.2 CDA治理结构

CDI 建立了**地方矿业发展委员会（Comité de Développement Local Minier, CDLM）**制度：

- CDLM 由地方行政长官（préfet régional）主持；
- 成员包括受影响社区代表、矿业公司代表、地方政府代表和公民社会代表；
- 负责管理地方矿业发展计划（Plan de Développement Local Minier）；
- 目前全国已建立 **9个** CDLM；
- 社区发展基金的拨付通过这9个地区的CDLM进行管理。

### 9.5.3 行业最佳实践模板

基于世界银行《矿业社区发展协议资料手册》和CDI本地经验，PMH的CDA应涵盖以下最佳实践内容：

**A. 核心条款**
- 明确界定"受影响社区"（communautés affectées / affected communities）的地理范围；
- 量化的社区发展投资承诺（超过法定最低0.5%）；
- 透明的基金管理和使用监督机制；
- 每5年一次的CDA审查和更新条款。

**B. 就业与培训**
- 本地雇员比例目标（建议：非技术岗位100%本地化，技术岗位逐年提升本地化比例）；
- 每年最低培训投资额度；
- 优先雇用受影响社区居民的明确承诺。

**C. 本地采购**
- 合格本地供应商数据库的建立和维护；
- 本地采购比例的年度目标和报告制度；
- 本地供应商能力建设计划。

**D. 环境保护与矿山关闭**
- 矿山关闭与复垦计划（Plan de Fermeture et de Réhabilitation）；
- 环境保证金（Garantie Environnementale）的缴纳安排；
- 环境监测数据的社区通报机制。

**E. 争端解决**
- 分层争端解决机制：协商→调解→仲裁；
- 设立常设协商委员会（Comité de Concertation Permanent）；
- 申诉处理流程和时限。

### 9.5.4 社区冲突案例及教训

**案例一：Divo金矿冲突（2016年）**

- **概况：** 2016年3月，居住在Divo金矿附近的居民与矿山运营方发生冲突；
- **根源：** 让出土地的居民认为获得的补偿与他们推测的矿山利润相比严重不足；
- **教训：** 补偿方案的制定须基于独立评估，并建立透明的信息沟通机制，避免因信息不对称导致社区对矿山利润的过度猜测。

**案例二：Agbaou矿持续紧张关系（2014-2018年）**

- **概况：** Agbaou矿管理层与当地居民代表之间长达4年的紧张关系和摩擦；
- **最终结果：** 2018年2月，双方终于达成协议；
- **教训：** 社区关系不能等问题爆发后再处理，须从项目初期即建立系统化的社区参与机制；早期投资于社区关系的成本远低于后期冲突处理的成本。

**案例三：CDI后冲突环境下的非法采矿**

- 2025年的研究揭示了CDI后冲突（post-conflict）环境下黄金开采活动中的治理挑战，包括非法采矿与合法运营之间的复杂互动关系（来源：[ScienceDirect - Gold Mining and Post-war Governmentality in CDI](https://www.sciencedirect.com/science/article/pii/S0305750X25001342)）；
- **教训：** PMH须评估其许可区域内是否存在非法手工采矿活动，并制定切实可行的应对策略，而非简单的对抗性做法。

## 9.6 对 PMH 的具体建议

### 9.6.1 3,000km² 目标的合规架构方案

**推荐方案：双SPV + 管理公司架构**

```
                    PMH（境外母公司）
                         │
          ┌──────────────┼──────────────┐
          │              │              │
     PMH CDI         PMH CDI        PMH CDI
    Management      Mining SPV-A   Mining SPV-B
      SARL            SA              SA
   （管理公司）    （采矿公司A）   （采矿公司B）
                      │              │
                 CDI政府10%      CDI政府10%
                 少数股东X%     少数股东Y%
                      │              │
                 持有许可：      持有许可：
                 6个小型 +       4个小型 +
                 1个大型         0个大型
                 ≈1,600km²      ≈1,400km²
```

**架构设计要点：**

| 要素 | SPV-A | SPV-B | Management SARL |
|------|-------|-------|-----------------|
| 法律形式 | Société Anonyme (SA) | Société Anonyme (SA) | SARL |
| PMH持股 | 85-90% | 85-90% | 100% |
| 政府Free Carry | 10% | 10% | 不适用 |
| 本地少数股东 | 不同于SPV-B | 不同于SPV-A | 不适用 |
| 独立董事会 | 是 | 是 | 是 |
| 独立银行账户 | 是 | 是 | 是 |
| 独立勘探计划 | 是 | 是 | 不适用 |
| 预计覆盖面积 | ≈1,600km² | ≈1,400km² | 不适用 |
| 功能 | 持有并运营许可 | 持有并运营许可 | 向两个SPV提供管理和技术服务 |

**为什么需要Management SARL：**
- 实现运营效率：两个SPV的日常管理、技术支持和行政事务由统一的管理公司提供；
- 通过管理服务费（management fees）实现集团内部资金调配的灵活性；
- 管理公司不直接持有矿业许可，因此不触发面积上限约束。

**两个SPV之间的实质性差异化设计：**
- **不同的少数股东：** SPV-A引入少数股东X（如CDI本地商业伙伴），SPV-B引入不同的少数股东Y；
- **不同的地理侧重：** SPV-A聚焦北部/东部区域，SPV-B聚焦南部/西部区域；
- **不同的矿种侧重（如适用）：** 如果许可覆盖不同矿种，可按矿种分配；
- **不同的管理层：** 虽然Hawk可同时担任两个SPV的顾问角色，但每个SPV应有不同的法定代表人（Directeur Général）。

### 9.6.2 10个小型许可证 + 1个大型许可证的最优持有分配

**推荐分配方案：**

| SPV | 持有许可 | 面积估算 | 理由 |
|-----|----------|----------|------|
| SPV-A | 6个小型 + 1个大型 | ≈1,600km² | 大型许可作为SPV-A核心资产，配合周边小型许可形成连续勘探区域 |
| SPV-B | 4个小型 | ≈1,400km² | 地理上独立于SPV-A的勘探区域集群 |

**分配原则：**
1. **地理连续性：** 同一SPV持有的许可应在地理上具有连续性或接近性，以体现独立的商业逻辑；
2. **避免人为拆分：** 不应将地理上相邻的、明显构成同一勘探项目的许可分配给不同SPV；
3. **大型许可的战略地位：** 大型许可是PMH在CDI的核心资产，应置于持有最强管理团队和最充裕资金的SPV-A中。

### 9.6.3 NSR 风险缓解策略

**立即行动（2026年Q1）：**

1. **全面审查** — Merlanih 须在30天内完成对11个许可证相关全部NSR合同的逐条审查，建立NSR权利登记表：

| 审查项目 | 内容 |
|----------|------|
| NSR费率 | 每个许可证的NSR比例 |
| NSR持有方 | 识别每个NSR的权利持有方 |
| 支付条件 | 触发支付的条件（勘探阶段 vs 生产阶段） |
| ROFR条款 | 是否存在优先购买权 |
| 否决权/同意权 | 是否需要NSR持有方同意进行某些交易 |
| 回购条款 | 是否存在回购选项及回购价格公式 |
| 审计权 | NSR持有方的审计权范围 |

2. **风险分级** — 将NSR合同按风险等级分类：
   - **高风险：** 含ROFR、否决权或无回购选项的合同；
   - **中风险：** 费率偏高（>1.5%）但无特殊限制条款的合同；
   - **低风险：** 费率≤1%且含回购选项的合同。

3. **策略执行：**
   - 对高风险合同：评估重新谈判或回购的可行性；
   - 对新获取许可：坚持NSR上限1.5%、无ROFR、含回购条款的标准条款；
   - 在生产阶段支付NSR时，采用Gross Revenue Royalty（GRR）计算方式而非NSR计算方式，以减少审计争议风险。
