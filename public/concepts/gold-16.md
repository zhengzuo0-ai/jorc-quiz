# 马里黄金 | Mali Gold Mining

> 💡 **马里是非洲第三大黄金生产国（仅次于加纳和南非），年产量约60-70吨。其Birimian绿岩带拥有世界级金矿——Barrick的Loulo-Gounkoto（Tier One资产）、B2Gold的Fekola（2023年全公司产量1,061,060 oz）、Resolute的Syama、Allied Gold的Sadiola。然而，2021年以来Assimi Goïta领导的军政府推行新2023年矿业法 (Code Minier 2023)，大幅强化资源民族主义政策，正在深刻重塑投资环境。估计超过100万手工采金矿工 (ASM/Orpaillage) 的庞大非正规部门更增添了复杂性。理解马里的地质优势与政治风险之间的张力，是评估西非黄金投资机会的关键。**

## 地质背景 (Geological Setting)

### 1. Birimian 绿岩带在马里 (Birimian Greenstone Belts in Mali)

马里的金矿化几乎全部赋存于古元古代 Birimian 绿岩带（约2.1-2.2 Ga），与加纳、布基纳法索、几内亚属同一成矿省。马里境内有三个主要的成矿区域：

**Kenieba 走廊 (Kenieba Window / Kenieba Inlier)**
- 位于马里西部，与塞内加尔接壤
- 主要矿体：**Loulo-Gounkoto** (Barrick Gold)、**Sadiola** (Allied Gold)、**Yatela**（已关闭）
- 岩性以基性-中性火山岩、浊积岩和侵入岩为主
- 金矿化受区域性剪切带控制——Senegal-Mali Shear Zone (SMSZ) 是最重要的构造
- 矿化类型：造山型金矿 (Orogenic Gold)，赋存于石英脉和蚀变带中

**Siguiri 盆地 (Siguiri Basin)**
- 从几内亚北部延伸进入马里南部
- 主要是沉积岩（浊积岩序列），褶皱和断裂发育
- **Syama** 矿区位于该构造域的延伸段
- 金矿化与铁质砂岩和碳质页岩中的硫化物（毒砂、黄铁矿）密切相关

**Kayes 地区 (Kayes Region)**
- 马里最西部的行政大区
- **Fekola** (B2Gold) 所在地——近年来西非最成功的金矿开发案例之一
- 矿化赋存于Birimian绿岩带的花岗闪长岩侵入体和周围变质岩接触带中
- 氧化矿延伸至地表以下80-100m，有利于低成本露天开采

### 2. 矿化类型与控矿因素 (Mineralization Types & Controls)

马里金矿以造山型金矿 (Orogenic Gold) 为主，主要控矿因素包括：

| 控矿因素 | 英文 | 说明 |
|----------|------|------|
| 剪切带 | Shear Zones | 区域性韧性-脆性剪切带是主要的流体通道和矿质沉淀场所 |
| 背斜转折端 | Anticlinal Hinges | 褶皱转折端处应力集中，有利于裂隙发育和流体汇聚 |
| 岩性接触带 | Lithological Contacts | 不同岩性的界面（如花岗岩-变质岩接触带）是渗透率突变带 |
| 侵入体边缘 | Intrusion Margins | 花岗闪长岩侵入体的边缘提供热源和流体驱动力 |
| 氧化富集 | Supergene Enrichment | 表生风化作用使浅部品位提高，形成氧化矿 (Oxide Ore) |

**氧化矿与原生矿的区别：** 马里许多矿床的浅部氧化矿（0-80m）品位更高且冶金回收率优异（>95%），因为金以自然金形式存在，可直接氰化浸出。深部原生矿 (Sulphide Ore) 中金与硫化物（毒砂 Arsenopyrite、黄铁矿 Pyrite）共生，需要更复杂的处理流程（浮选+氧化预处理），回收率通常为85-92%。

## 主要矿山 (Major Mines)

### 3. Loulo-Gounkoto 综合体 (Barrick Gold) — Tier One 资产

**Loulo-Gounkoto** 是马里最大的金矿运营综合体，被Barrick Gold列为其全球五大 **Tier One** 资产之一（Tier One定义：年产量>500,000 oz、矿山寿命>10年、下半成本曲线AISC）。

| 参数 | 数据 |
|------|------|
| 运营商 | Barrick Gold (80%), 马里政府 (20%) |
| 年产量 | ~550,000-680,000 oz Au |
| 矿山类型 | 地下 + 露天联合开采 |
| AISC | ~$1,000-1,200/oz |
| 矿山寿命 | 至2030年代中期 |
| 处理厂产能 | ~4.7 Mtpa |
| 战略地位 | Barrick全球Tier One资产；马里最大单一黄金运营 |

Loulo矿区包含多个矿体——Yalea和Gara是两个主要的地下矿山，采用长壁式崩落法 (Sublevel Caving) 和上向进路充填法 (Underhand Cut and Fill)。Gounkoto是露天矿，2021年开始考虑地下延伸。该综合体的成功关键在于高品位地下矿石（5-8 g/t Au）与低品位露天矿石（2-3 g/t Au）的混合配比，优化了处理厂的给矿品位。

**政治风险注意：** Barrick在马里面临军政府施压——2024年多名Barrick员工被拘留，税务争议持续。尽管如此，Loulo-Gounkoto的Tier One地位使其成为Barrick难以放弃的核心资产。

### 4. Fekola (B2Gold) — 旗舰矿山与政治挑战并存

**Fekola** 是B2Gold的旗舰矿山，2017年投产以来表现卓越。B2Gold 2023年全公司总产量为 **1,061,060 oz**（来源：B2Gold 2023 Annual Report），Fekola是最大贡献者。

| 参数 | 数据 |
|------|------|
| 运营商 | B2Gold (80%), 马里政府 (20%) |
| 年产量 | ~550,000-600,000 oz Au（Fekola主矿区） |
| B2Gold全公司产量 | **1,061,060 oz**（2023年实际） |
| 矿山类型 | 露天开采；**2025年获批地下开采** |
| AISC | ~$800-1,000/oz |
| 处理厂产能 | ~8 Mtpa（2022年扩建后） |
| 剥采比 | ~5-7:1 (waste:ore) |

Fekola的竞争优势在于浅部高品位氧化矿（早期品位>3 g/t Au）和大规模的矿体规模。2022年完成的处理厂扩建（从5 Mtpa至8 Mtpa）使其成为西非最大的金矿之一。然而，随着开采深度增加，矿石类型从氧化矿过渡至原生矿，品位和回收率面临下降压力。B2Gold在Fekola周边持续开展区域勘探，2021年发现的Anaconda区域延伸了矿山寿命。

**关键进展 (Key Developments)：**

- **Fekola Regional 许可证受阻：** 军政府延迟颁发Fekola Regional开采许可证，导致 **80,000-100,000 oz/yr 的产量损失**。这是资源民族主义对运营产生直接影响的典型案例
- **2025年地下开采获批：** B2Gold获得了Fekola地下开采的审批，将解锁深部硫化矿资源，延长矿山寿命
- **稳定条款保护 (Stabilization Clause)：** Fekola的矿业协议 (Mining Convention) 包含稳定条款，保护现有协议免受新2023年矿业法 (Code Minier 2023) 追溯条款的影响。这是投资者评估马里资产时的关键法律保护机制——已签署矿业协议的稳定条款与新法律的追溯适用性之间的张力，是当前马里矿业法律争议的核心

### 5. Syama (Resolute Mining)

**Syama** 是全球首个全自动化地下金矿，位于马里南部Sikasso地区。

| 参数 | 数据 |
|------|------|
| 运营商 | Resolute Mining (80%), 马里政府 (20%) |
| 年产量 | ~200,000-250,000 oz Au |
| 矿山类型 | 地下（自动化SubLevel Caving）+ 氧化矿露天 |
| 矿石特征 | 难处理硫化物矿石（Refractory Sulphide Ore） |
| 处理工艺 | 焙烧 (Roasting) + CIL |

Syama的矿石是典型的难处理型——金细粒包裹在毒砂和黄铁矿中，必须先通过焙烧（约650°C）氧化硫化物，释放金颗粒后再进行常规氰化浸出。Resolute于2018年引入了Sandvik的AutoMine系统，实现了地下采矿的高度自动化。这一技术突破降低了人工成本，但维护和技术支持的复杂性也相应增加。

### 6. Sadiola

**Sadiola** 矿山位于Kenieba走廊北端，运营历史超过20年。

- 原由AngloGold Ashanti和IAMGold合资运营
- 2020年Allied Gold收购，开启硫化物矿体开发项目（Sadiola Sulphide Project）
- 氧化矿已近枯竭，未来依赖深部硫化物矿石——需要建设新的处理设施
- 硫化物处理厂采用CIL + 浮选工艺

## 矿业法与政治风险 (Mining Code & Political Risk)

### 7. 矿业法演变：从2019到2023 (Mining Code Evolution: 2019 → 2023)

马里于2019年颁布矿业法，但军政府在2023年推出了更为激进的 **新2023年矿业法 (Code Minier 2023)**，大幅强化了国家对矿业资源的控制。两部法律的对比体现了资源民族主义的升级：

**2019年矿业法 (原法)：**

| 条款 | 内容 |
|------|------|
| 政府持股 | 免费持有10%，可增购至20% |
| 税收稳定期 | 勘探许可证有效期内税收政策不变 |
| 企业所得税 | 25-30%（取决于矿种和阶段） |
| 特许权使用费 (Royalty) | 3-7%（按金价滑动计算） |
| 本地采购要求 | 优先使用马里本地供应商和劳动力 |
| 环境基金 | 矿企须按产值比例向环境修复基金缴费 |
| 出口限制 | 黄金出口须经中央银行审批 |

**2023年矿业法 (新法) — 关键变化：**

| 条款 | 2023年新规 | 影响评估 |
|------|-----------|----------|
| 政府免费持股 | 从10%大幅提升至 **最高35%** | 直接稀释外资矿企权益 |
| 追溯适用 | 军政府主张新法适用于现有矿业协议 | 与稳定条款产生法律冲突 |
| 本地化要求 | 更严格的本地采购、本地雇员、本地加工要求 | 增加运营成本和合规负担 |
| 出口与税务 | 加强黄金出口管控和税务审计权 | 资金汇回风险上升 |

**稳定条款的法律博弈：** 已签署矿业协议（如B2Gold Fekola）中的稳定条款 (Stabilization Clause) 理论上保护现有投资免受新法追溯影响。但军政府的执法实践与法律承诺之间存在显著差距，这是当前马里矿业法律争议的核心问题。

### 8. 军事政变与资源民族主义 (Military Junta & Resource Nationalism)

马里自2020年8月和2021年5月经历两次军事政变后，**Assimi Goïta上校领导的军政府（自2021年至今执政）** 对矿业的影响深远且持续升级：

**直接影响 — 已发生的事件：**
- **2023年：** 马里政府以税务审计为由**逮捕了Resolute Mining的CEO**，施压要求支付约1.6亿美元的"税务调整"
- **2024年：** **Barrick Gold多名员工被拘留**，税务争议持续
- **2023年：** 军政府颁布**新2023年矿业法 (Code Minier 2023)**，要求将政府免费持股比例从10%提高至**最高35%**
- **B2Gold Fekola Regional许可证被延迟**，直接导致80,000-100,000 oz/yr产量损失
- 退出ECOWAS，西方制裁增加了进出口和资金汇回的复杂性

**深层逻辑：**
- 马里政府财政高度依赖矿业收入（黄金占出口的70%+）——马里是**非洲第三大黄金生产国**（仅次于加纳和南非）
- 军政府需要额外资金支持北部反恐军事行动和社会支出
- 布基纳法索和尼日尔的军政府采取了类似的资源民族主义策略，三国结成**萨赫勒联盟 (Alliance of Sahel States)**，形成区域趋势
- 中俄企业（如Wagner/Africa Corps关联实体）在马里的影响力上升，改变了地缘政治格局
- 军政府将矿业收入最大化视为政权合法性的核心支柱

**投资者应对策略：**
- 国际仲裁条款（ICSID）和双边投资保护协定 (BIT) 是最后防线
- 多数运营商选择协商妥协而非对抗——"付税保运营"是现实选择
- 政治风险保险 (MIGA/PRI) 的重要性凸显
- 部分投资者重新评估马里资产的折现率——从8-10%提升至**12-15%+**
- **稳定条款 (Stabilization Clause)** 的法律效力成为核心保护机制——如B2Gold Fekola的矿业协议

## ASM 行业现实 (ASM Sector Reality)

### 9. 手工小矿开采 (Artisanal & Small-Scale Mining)

马里估计有**超过100万手工采金矿工** (estimated 1M+ artisanal miners) 直接或间接从事ASM活动（当地称为 **Orpaillage**），使马里成为非洲ASM规模最大的国家之一。活动主要集中在Kayes、Sikasso和Koulikoro地区。

**ASM的特征：**
- 年产量估计为20-40吨，占马里黄金总产量的30-40%——但大部分未进入官方统计
- 开采方法以浅部（0-30m）手工挖掘竖井和原始碾磨为主
- 汞 (Mercury) 使用广泛——用于汞齐法提金，造成严重环境和健康问题
- 多数ASM活动在法律灰色地带——缺乏正式许可，但政府默许其存在
- 儿童劳动问题严重——国际劳工组织估计约20,000名儿童参与ASM

**ASM与工业矿山的冲突：**
- ASM矿工常进入工业采矿许可证范围内开采，导致安全隐患和法律纠纷
- 工业矿企需在社区关系 (Community Relations) 上投入大量资源
- 部分矿企采取"共存策略"——划定ASM许可区域，提供技术培训和替代生计项目
- Barrick在Loulo-Gounkoto附近建立了社区发展基金，B2Gold在Fekola周边实施了农业替代项目

**ASM供应链：**
- ASM黄金通过多层中间商流向巴马科 (Bamako) 的贸易商
- 迪拜是马里ASM黄金的主要出口目的地——通过非正式渠道出口
- 负责任矿产采购（LBMA Responsible Sourcing）对ASM供应链构成挑战
- 部分国际组织（如Alliance for Responsible Mining）推动ASM正规化

## PMH 马里 Track B 现金流运营 (PMH Mali Track B Cash Flow Operations)

### 10. Track B 运营模式

PMH的Track B策略聚焦于通过小规模合规化运营在马里产生早期现金流。在当前军政府治理和新2023年矿业法环境下，Track B运营面临独特的机遇与风险：

**运营模式要点：**
- 与持有合法许可证的马里本地合作伙伴建立合资关系
- 利用半机械化方法（小型挖掘机、便携式碾磨设备、重力选矿）处理高品位矿石
- 避免使用汞——采用重力精选 + 浮选 + 小型CIL的清洁工艺链
- 目标产量：2,000-5,000 oz/年，作为早期现金流来源

**关键风险管理（2024-2025更新）：**
- **合规框架：** 需同时满足2019年矿业法和新2023年矿业法的许可要求——法律环境处于过渡期，合规成本上升
- **资金流转：** 军政府加强外汇管控，资金汇回是核心挑战——需要与当地银行建立稳定的美元/CFA兑换渠道
- **政府持股风险：** 新矿业法要求最高35%政府免费持股，直接影响小型运营的股权结构设计
- **社区许可 (Social License to Operate)：** 与当地村庄和传统酋长 (Chef de Village) 建立正式协议；在100万+ ASM矿工的环境中导航
- **安全保障：** 北部和中部安全局势持续恶化，运营需集中在南部相对稳定的地区
- **环境合规：** 小型运营也须提交环境影响评估 (ESIA) 并缴纳环境修复保证金

**现金流模型：**
- 典型的Track B运营在金价$2,000/oz时的操作利润率约为30-40%
- 盈亏平衡点约为$1,200-1,400/oz AISC
- 资本投入：$500K-2M用于设备和基础设施
- 回收期目标：12-18个月
- **注意：** 在当前政治风险溢价下，投资者对马里小型运营的要求回报率已显著提升
