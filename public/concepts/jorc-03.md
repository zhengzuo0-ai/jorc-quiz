# 资源量与储量分类 | Resource vs Reserve Framework

> 💡 **矿产资源量和矿石储量是两个不同维度的概念：资源量衡量"地下有多少矿"，储量衡量"能经济开采出多少矿"——两者之间的差距，是矿业投资中最关键的认知。**

## 核心概念 (Core Concepts)

### 1. JORC分类框架全景 (JORC Figure 1)

JORC准则的分类框架可以用两个维度来理解：**地质确信度 (geological confidence)** 和 **经济可行性 (economic viability)**。

```
                    地质确信度增加 ──────────────────────→

 勘探结果             矿产资源量                    矿石储量
(Exploration    ┌─────────────────────────┐   ┌───────────────────┐
 Results)       │                         │   │                   │
                │  推断资源量              │   │                   │
    ──→         │  (Inferred Resource)     │   │                   │
                │         │                │   │                   │
                │         ▼                │   │                   │
                │  指示资源量 ─────────────────→ 概略储量          │
                │  (Indicated Resource)    │   │ (Probable Reserve)│
                │         │                │   │       │           │
                │         ▼                │   │       ▼           │
                │  探明资源量 ─────────────────→ 证实储量          │
                │  (Measured Resource)     │   │ (Proved Reserve)  │
                │                         │   │                   │
                └─────────────────────────┘   └───────────────────┘
                                                       ▲
                                              修正因子的正面评估
                                              (Modifying Factors)
```

**关键理解**：
- 左侧（资源量）是纯地质维度——"我们有多大把握说地下有矿？"
- 右侧（储量）加入了经济和技术维度——"这些矿能赚钱地挖出来吗？"
- 从资源量到储量的跨越，需要经过**修正因子 (Modifying Factors)** 的综合评估

### 2. 矿产资源量的三个级别

#### 推断资源量 (Inferred Mineral Resource)

**定义**：基于有限的地质证据和采样，可以合理推断矿化的存在和品位，但地质连续性尚未得到充分证实。

**特征**：
- 钻探间距较宽（通常100米以上，视矿床类型而定）
- 矿体的形状、延伸和连续性主要依赖地质推断而非直接证据
- 品位和吨位估算的不确定性较高（可能有 +/-50% 甚至更大的误差）
- **不能直接转化为矿石储量**——这是一个关键规则

**能做什么**：
- 用于内部矿山规划的概念性评估
- 用于概略研究 (Scoping Study) 的初步经济评价（但必须注明局限性）
- 作为指导进一步勘探投资的依据

**不能做什么**：
- 不能用于预可行性研究 (PFS) 或可行性研究 (FS) 中的储量转换
- 不能作为矿山投资决策的主要依据
- 在财务报表中需要极其谨慎地引用

#### 指示资源量 (Indicated Mineral Resource)

**定义**：基于充分的地质证据和采样，矿化的存在、品位、连续性和形状可以合理估算。

**特征**：
- 钻探间距适中（通常50-100米，视矿床类型而定）
- 矿体连续性已通过多个数据点得到合理验证
- 品位和吨位估算有合理的精度（典型不确定性 +/-25-30%）
- **可以转化为概略储量 (Probable Reserve)**

**从Inferred到Indicated需要什么？**

| 要素 | 具体要求 |
|------|---------|
| 更密的钻探间距 | 将钻探网格从100m+缩小到50-100m |
| 地质连续性证据 | 多个截面和平面上的矿化对比，证明矿体在三维空间中是连续的 |
| 更多的QA/QC数据 | 重复样、标准样、空白样的系统性检验 |
| 改进的地质模型 | 矿化控制因素（构造、岩性、蚀变）的更好理解 |
| 合理的变差函数 | 地质统计分析表明数据支持空间连续性假设 |

#### 探明资源量 (Measured Mineral Resource)

**定义**：基于详细和可靠的地质证据和采样，矿化的存在、品位、连续性和形状可以高度确信地估算。

**特征**：
- 钻探间距很小（通常25-50米或更小）
- 矿体在三维空间中的几何形态已被精确刻画
- 品位和吨位估算具有高精度（典型不确定性 +/-10-15%）
- **可以转化为证实储量 (Proved Reserve)**

**从Indicated到Measured需要什么？**

| 要素 | 具体要求 |
|------|---------|
| 极密的钻探网格 | 间距进一步缩小到25-50m |
| 详细的地质模型 | 三维矿体模型的高度精确性，所有地质边界都有多个控制点 |
| 高质量的数据 | 全面的QA/QC程序、独立验证、多种分析方法交叉验证 |
| 矿化连续性确认 | 变差函数的高质量拟合，克里金方差在可接受范围内 |
| 密度/体积数据 | 充足的密度测定，以准确估算吨位 |

### 3. 矿石储量的两个级别

#### 概略储量 (Probable Ore Reserve)

**来源**：主要由指示资源量 (Indicated Resource) 通过修正因子评估后转化。在特殊情况下，探明资源量 (Measured Resource) 也可以因为修正因子的不确定性而仅转化为概略储量。

**含义**：在当前和可预见的经济、技术条件下，该部分矿产可以经济地开采。但由于数据的局限性，仍存在一定的不确定性。

#### 证实储量 (Proved Ore Reserve)

**来源**：由探明资源量 (Measured Resource) 通过修正因子评估后转化。

**含义**：具有最高级别的确信度——无论是地质方面还是经济技术方面，都已经过严格验证。这是矿山融资、项目估值和生产计划的最可靠依据。

### 4. 修正因子 (Modifying Factors) — 从资源到储量的桥梁

修正因子是将"地下有多少矿"转化为"能经济采出多少矿"的关键参数。主要包括：

| 修正因子 | 具体内容 | 示例 |
|---------|---------|------|
| 采矿因子 (Mining Factors) | 采矿方法、回收率、贫化率、采矿损失 | 地下采矿回收率可能只有75-85%，而露天采矿可达95%+ |
| 冶金因子 (Metallurgical Factors) | 选矿回收率、流程设计、精矿品质 | 金矿氰化浸出回收率通常90-95%，但某些难处理矿石可能仅70% |
| 经济因子 (Economic Factors) | 商品价格、运营成本、资本开支、折现率 | 金价$1,800/oz时经济的矿石，在$1,200/oz时可能不经济 |
| 基础设施 (Infrastructure) | 道路、电力、水源、尾矿库 | 偏远地区的基础设施成本可能使项目不经济 |
| 市场因子 (Marketing Factors) | 精矿销售合同、运输、港口 | 锂精矿的市场可达性可能影响项目价值 |
| 法律与环保 (Legal & Environmental) | 采矿权、环评批准、社区许可 | 无采矿许可证的矿产不能计为储量 |
| 社会与政府 (Social & Government) | 土地所有权、原住民权益、政府政策 | 社区反对可能阻止项目开发 |

**关键认知**：修正因子中的任何一项出现负面结论，都可能导致资源量无法转化为储量。一个地质上品位很高的矿体，如果位于国家公园内或没有水资源，储量就可能为零。

### 5. CRIRSCO国际对标

全球主要矿产报告标准在分类体系上高度一致，都在CRIRSCO框架下：

| JORC (澳大利亚) | CIM (加拿大) | SAMREC (南非) | SME (美国) | PERC (欧洲) |
|----------------|------------|-------------|-----------|------------|
| Inferred Resource | Inferred Resource | Inferred Resource | Inferred Resource | Inferred Resource |
| Indicated Resource | Indicated Resource | Indicated Resource | Indicated Resource | Indicated Resource |
| Measured Resource | Measured Resource | Measured Resource | Measured Resource | Measured Resource |
| Probable Reserve | Probable Reserve | Probable Reserve | Probable Reserve | Probable Reserve |
| Proved Reserve | Proved Reserve | Proved Reserve | Proven Reserve | Proved Reserve |

**虽然名称几乎相同，但细节有差异**：
- 美国SEC S-K 1300于2021年实施，首次与CRIRSCO对齐——此前美国的矿产报告标准与国际标准有显著差异
- 加拿大NI 43-101对"预可行性研究"和"可行性研究"有更详细的定义
- 各标准对修正因子的具体要求略有不同

### 6. 核心误区：Resources ≠ Reserves

**这是投资者最常犯的错误——将矿产资源量 (Mineral Resources) 等同于矿石储量 (Ore Reserves)。**

区别的本质：

| 维度 | 矿产资源量 (Resources) | 矿石储量 (Reserves) |
|------|---------------------|-------------------|
| 定义 | 地壳中已知或推断存在的矿化体 | 经技术和经济评估后可盈利开采的部分 |
| 经济考量 | 仅需"合理的最终经济开采前景" | 必须通过完整的经济可行性评估 |
| 包含关系 | 储量是资源量的子集（储量 ⊂ 资源量） | — |
| 投资意义 | 表明项目的"潜力上限" | 表明项目的"可实现价值" |
| 融资适用性 | 银行通常不会基于资源量提供项目融资 | 银行项目融资通常仅基于证实和概略储量 |

### 7. 典型转换率

从资源量到储量的转换率因矿种、矿床类型和项目条件而异：

| 矿种 | 典型转换率 (Indicated → Probable) | 影响因素 |
|------|----------------------------------|---------|
| 金（露天） | 70-85% | 品位截断、剥采比、回收率 |
| 金（地下） | 60-75% | 采矿方法、支护成本、通风 |
| 铁矿 | 80-90% | 通常简单几何形态，高转换率 |
| 铜斑岩 | 70-85% | 大规模、低品位，规模经济 |
| 锂（硬岩） | 65-80% | 选矿复杂性、市场价格波动 |
| 煤 | 75-90% | 相对简单的开采条件 |

**转换率低于60%的常见原因**：
- 矿体形态复杂，采矿损失高
- 冶金回收率低于预期
- 基础设施成本过高
- 环保或社区审批困难
- 商品价格假设过于保守

### 8. 分类路径的实际工作流

一个典型的矿产项目从发现到储量确认的时间线：

```
第1-2年  草根勘探 → 勘探结果 (Exploration Results)
         ├─ 地质填图、地球化学采样、物探
         └─ 发现矿化异常，开始钻探

第2-4年  系统钻探 → 推断资源量 (Inferred Resource)
         ├─ 宽间距钻探（100m+），初步地质模型
         └─ 概略研究 (Scoping Study)

第4-6年  加密钻探 → 指示资源量 (Indicated Resource)
         ├─ 中等间距钻探（50-100m），改进地质模型
         └─ 预可行性研究 (Pre-Feasibility Study, PFS)

第5-8年  进一步加密 → 探明资源量 (Measured Resource)
         ├─ 密集钻探（25-50m），详细地质模型
         └─ 可行性研究 (Feasibility Study, FS)

第6-10年 修正因子评估 → 矿石储量 (Ore Reserves)
         ├─ 采矿设计、冶金测试、经济分析
         └─ 建设决策 (Construction Decision)
```

**注意**：这是一个典型时间线，实际进度因项目而异。某些大型项目从发现到生产可能需要15年以上。

## PMH 实践关联 (PMH Context)

> **资源量与储量分类对PMH的直接意义：**
>
> - **项目估值**：PMH的市值在很大程度上取决于其资源量和储量的规模与级别。投资者对探明储量 (Proved Reserve) 的估值倍数远高于推断资源量 (Inferred Resource)。每一次成功的资源量升级（如从Inferred升至Indicated），通常会带来股价的正面反应。
> - **融资能力**：当PMH寻求项目融资 (project finance) 时，银行通常只认可概略储量 (Probable) 和证实储量 (Proved) 作为贷款基础。推断资源量不被银行接受。这意味着PMH需要在融资前将资源量升级到足够高的级别。
> - **西非勘探策略**：PMH在西非的金矿勘探项目，钻探间距的选择直接决定资源分类级别。在勘探预算有限的情况下，如何在扩展矿化范围（增加推断资源量的吨位）和加密钻探（提升资源量级别）之间取得平衡，是关键的战略决策。
> - **资源量到储量的转化**：PMH的项目要从资源量转化为储量，需要完成修正因子评估。对于西非项目，基础设施（道路、电力、水源）和社区许可可能是关键的修正因子瓶颈。
> - **国际报告标准**：PMH如果考虑在加拿大TSX或其他交易所双重上市，需要了解JORC分类与CIM/NI 43-101的对应关系，确保资源量声明在不同标准间的一致性。
