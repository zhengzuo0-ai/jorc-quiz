# 资源量与储量分类 Resource & Reserve Classification

## JORC 分类框架 Classification Framework

### 矿产资源量 Mineral Resources

矿产资源量是指在地壳中或地表上，以一定形式、品位(或质量)和数量存在的，**有合理前景被最终经济开采**的固体矿物的集中体或产出体。

#### 推断矿产资源量 Inferred Mineral Resource
- **地质确定性最低** — 基于有限的地质证据和取样
- 品位和/或质量的估计是根据有限的信息推断的
- 地质连续性可以合理假设但不能确认
- **不能直接转换为储量**
- 用途: 初步概念研究，不能用于可行性研究

#### 标示矿产资源量 Indicated Mineral Resource
- **中等地质确定性** — 足以进行有意义的经济评价
- 基于从适当间距的勘探手段获得的信息
- 矿化的品位、吨位、形态、密度可以合理估计
- **可以转换为概略储量 (Probable Reserve)**
- 用途: 预可行性研究

#### 探明矿产资源量 Measured Mineral Resource
- **最高地质确定性** — 品位和吨位可以可靠估计
- 基于详细和可靠的勘探、取样和测试信息
- 矿化的品位、吨位、形态、密度确定性高
- **可以转换为证实储量 (Proved Reserve)**
- 用途: 可行性研究

### 矿石储量 Ore Reserves

矿石储量是指矿产资源中经过适当的修正因素评估后，**可以经济开采**的部分。

#### 概略矿石储量 Probable Ore Reserve
- 从**标示资源量** (有时从探明资源量) 转换而来
- 经过修正因素评估
- 经济可行性在预可行性研究级别已得到验证
- 信心水平低于证实储量

#### 证实矿石储量 Proved Ore Reserve
- 仅从**探明资源量**转换而来
- 经过最详细的修正因素评估
- **最高信心水平**
- 常用于银行融资和矿山设计

## 修正因素 Modifying Factors

从资源量到储量的转换需要考虑的因素:

| 修正因素 | 考虑内容 | 影响 |
|---|---|---|
| 采矿 Mining | 开采方法、回收率、贫化 | 损失15-30%矿石 |
| 冶金 Metallurgy | 选矿回收率、工艺 | 10-20%金属损失 |
| 经济 Economic | 金属价格、运营成本、资本成本 | 低品位部分被排除 |
| 市场 Marketing | 产品销售、合同 | — |
| 法律 Legal | 采矿权、许可证 | 限制范围 |
| 环境 Environmental | 环评、排放标准 | 限制方法/范围 |
| 社会 Social | 社区关系、土地使用 | 限制范围 |
| 政府 Governmental | 税收、政策 | 影响经济性 |

### 转换关系图
```
探明资源 (Measured) → 证实储量 (Proved)     [最高确定性]
标示资源 (Indicated) → 概略储量 (Probable)   [中等确定性]
推断资源 (Inferred) → ✗ 不能直接转换储量      [不足以支撑经济评估]

注: 探明资源也可以转换为概略储量 (如果修正因素不确定性较大)
```

## 分类依据 Classification Criteria

### 钻探间距与分类 Drill Spacing & Classification

| 矿床类型 | 推断 Inferred | 标示 Indicated | 探明 Measured |
|---|---|---|---|
| 均匀矿化 (斑岩) | 100-200m | 50-100m | 25-50m |
| 中等变化 (脉状) | 50-100m | 25-50m | 10-25m |
| 高变化 (砂金) | 25-50m | 10-25m | 5-10m |

*注: 这些只是一般指导，实际分类需要根据变差函数和数据质量综合判断。*

### 其他分类考虑因素
1. **数据质量** — QA/QC结果、取样方法
2. **地质连续性** — 矿化在钻孔之间是否可追踪
3. **变差函数** — 空间相关性范围
4. **克里金方差** — 估算的不确定性
5. **对比验证** — 产量数据与估算的对比

---

*核心区别: Resource = 有地质依据 + 有合理经济前景; Reserve = Resource + 经过修正因素验证可经济开采。信心递增: Inferred → Indicated → Measured → Probable → Proved*
