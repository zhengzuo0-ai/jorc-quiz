# 矿业经济学 Mining Economics

## 矿业项目评估 Project Evaluation

### 可行性研究阶段 Feasibility Study Stages

| 阶段 Stage | 精度 Accuracy | 成本估算范围 | 目的 |
|---|---|---|---|
| 概略研究 Scoping Study | ±50% | 初步概念 | 值得继续吗？ |
| 预可行性 Pre-Feasibility (PFS) | ±25% | 初步设计 | 项目是否可行？ |
| 可行性 Feasibility (DFS/BFS) | ±15% | 详细设计 | 项目是否能融资？ |

### 关键经济指标 Key Economic Metrics

#### NPV 净现值 Net Present Value
```
NPV = Σ [CFₜ / (1 + r)ᵗ] - Initial Investment
```
- CF = 现金流, r = 折现率, t = 年份
- NPV > 0 → 项目可行
- 矿业常用折现率: 5-10%

#### IRR 内部收益率 Internal Rate of Return
- 使 NPV = 0 的折现率
- IRR > 公司要求回报率 → 项目可行
- 矿业最低IRR通常: 15-25%

#### Payback Period 投资回收期
- 累计现金流变为正数的时间
- 矿业通常要求 3-5年

### 成本结构 Cost Structure

#### CAPEX 资本支出 Capital Expenditure
| 项目 Item | 占比 |
|---|---|
| 采矿设备 Mining Equipment | 20-30% |
| 选矿厂 Processing Plant | 30-40% |
| 基础设施 Infrastructure | 15-25% |
| 其他 (环评、管理) | 10-20% |

典型金矿CAPEX: **$200M - $2B** (取决于规模)

#### OPEX 运营成本 Operating Expenditure
| 项目 Item | 占比 |
|---|---|
| 采矿 Mining | 30-40% |
| 加工 Processing | 25-35% |
| 一般管理 G&A | 10-15% |
| 其他 (运输、环保) | 10-15% |

#### AISC 全维持成本 (All-In Sustaining Cost)
```
AISC = Cash Cost + Sustaining Capex + G&A + Exploration + Reclamation
```
- 2024全球平均: **$1,000-1,300/oz**
- 低成本矿: <$900/oz
- 高成本矿: >$1,500/oz

### 敏感性分析 Sensitivity Analysis

NPV对以下因素最敏感:
1. **金属价格** — 影响最大
2. **品位 Grade** — 直接影响收入
3. **回收率 Recovery** — 影响产出
4. **运营成本 OPEX** — 影响利润
5. **资本支出 CAPEX** — 影响初始投资

## 矿业融资 Mining Finance

### 融资方式 Financing Methods

| 方式 Method | 描述 | 优势 | 劣势 |
|---|---|---|---|
| 股权融资 Equity | 发行股票 | 无还债压力 | 稀释股权 |
| 债务融资 Debt | 银行贷款 | 不稀释 | 利息负担 |
| 流媒体 Streaming | 预付换折价购买权 | 快速资金 | 长期让利 |
| 特许权 Royalty | 按收入%支付 | 不参与运营 | 永久性义务 |
| 项目融资 Project Finance | 以项目现金流担保 | 不追溯母公司 | 要求高确定性 |
| 合资 Joint Venture | 多方共担 | 分散风险 | 决策复杂 |

### 关键融资门槛
- 银行通常要求: **证实+概略储量 (Proved + Probable)** 支持的矿山寿命
- 资源量 (尤其是推断资源量) 不能用于融资
- 需要可行性研究 (DFS) 级别的报告

---

*核心公式: NPV判断是否值得投; IRR判断回报是否足够; AISC判断是否能盈利; 敏感性分析判断最大风险在哪。*
