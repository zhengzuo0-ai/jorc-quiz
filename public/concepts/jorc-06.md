# 矿产资源估算 Mineral Resource Estimation

## 估算流程 Estimation Workflow

### 1. 数据编汇 Data Compilation
- 钻孔数据库建立 (collar, survey, assay, geology)
- 数据验证 (查重、逻辑检查、拓扑检查)
- QA/QC 评估

### 2. 地质建模 Geological Modelling
- **矿体解释 (Interpretation)**: 在截面和平面上圈定矿化边界
- **3D线框模型 (Wireframe)**: 连接截面形成三维矿体模型
- **领域 (Domains)**: 按地质特征划分不同估算域
  - 氧化带 vs 硫化带
  - 高品位脉 vs 低品位晕
  - 不同岩性/矿化类型

### 3. 统计分析 Statistical Analysis
- **基本统计**: 均值、中位数、标准差、变异系数 (CV)
- **直方图**: 品位分布特征
  - 正态分布 → 普通克里金适用
  - 对数正态 → 可能需要特殊处理
  - 双峰分布 → 需要分域处理
- **异常值处理 (Top-cutting/Capping)**
  - 方法: 概率图断点、对数概率图、CV分析
  - 目的: 减少少数高品位样品对估算的过度影响

### 4. 变差函数分析 Variogram Analysis

**变差函数**描述品位在空间上的相关性:

```
γ(h) = (1/2N) × Σ[Z(xᵢ) - Z(xᵢ+h)]²
```

**关键参数:**
| 参数 | 含义 | 应用 |
|---|---|---|
| 块金值 Nugget (C₀) | 零距离方差 (随机变异) | 反映取样误差和微观不均匀 |
| 基台值 Sill (C₀+C) | 总方差 | 数据总变异性 |
| 变程 Range (a) | 空间相关距离 | **决定钻探间距和搜索半径** |
| 块金效应 Nugget Ratio | C₀/(C₀+C) | >50% 表示数据高度不均匀 |

**变差函数模型:**
- 球状模型 Spherical: 最常用
- 指数模型 Exponential: 无限接近基台
- 高斯模型 Gaussian: 近原点连续性好

### 5. 块体模型 Block Model
- 将矿体划分为规则的三维块体
- 典型尺寸: 5m × 5m × 5m 到 20m × 20m × 10m
- 块体大小应反映: 选矿最小开采单元 (SMU)

### 6. 品位插值 Grade Interpolation

#### 普通克里金 Ordinary Kriging (OK)
- 最常用的方法
- 假设: 平稳随机函数，局部均值未知
- 使用变差函数确定权重
- 提供**估算值和估算方差 (kriging variance)**

#### 反距离加权 Inverse Distance Weighting (IDW)
- 简单直观
- 权重 = 1/d^n (n通常取2或3)
- 不需要变差函数
- 不提供估算方差

#### 多指标克里金 Multiple Indicator Kriging (MIK)
- 在多个品位阈值上建模
- 适合**高度偏斜的金品位分布**
- 提供品位概率分布

### 7. 验证 Validation
- **目视检查**: 截面图、平面图对比钻孔数据和块体品位
- **全局对比**: 块体模型均值 vs 钻孔数据均值
- **Swath plots**: 沿各方向条带对比模型和数据
- **品位-吨位曲线**: 不同截断品位下的吨位和品位变化

### 8. 资源量分类
基于: 钻探间距、数据质量、变差函数参数、克里金方差

---

*估算流程: 数据→地质模型→统计→变差函数→块体模型→插值→验证→分类。每一步都环环相扣。*
