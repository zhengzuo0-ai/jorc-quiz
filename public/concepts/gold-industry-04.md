# 马里矿业法规详解 Mali Mining Code Deep Dive

*数据来源: [UNCTAD Investment Policy Hub](https://investmentpolicy.unctad.org/investment-policy-monitor/measures/4352/mali-adopts-new-mining-code-increasing-the-state-and-local-participation-in-new-projects-), [Lexology - Mali Mining Compliance](https://www.lexology.com/library/detail.aspx?g=d34f08f2-c46a-4d50-8669-932a49dc1a4a), [LALIVE](https://www.lalive.law/the-impact-of-malis-revised-legislation-on-foreign-mining-compliance/), [IIED](https://www.iied.org/malis-new-mining-law-improvement-fails-artisanal-miners), [South Centre](https://www.southcentre.int/wp-content/uploads/2025/04/SV287_250428.pdf)*

## 背景 Context

> **为什么这很重要?** 马里是非洲第三大黄金生产国（年产约100吨），但2020-2021年连续两次军事政变后，政府大幅收紧了矿业政策。2023年新矿业法彻底改变了游戏规则——外国公司的持股比例被压缩，税负大幅增加。**任何想在马里投资矿业的人，必须深入理解这些变化。**

## 2023年新矿业法 New Mining Code (2023年9月1日生效)

### 颁布背景
- 2022年马里政府对黄金矿业进行审计
- 发现: 国家因不利合同、税收漏洞和监管不力，损失了约 **CFA 3,000-6,000亿** (约$4.8-9.6亿美元)
- 政府随后追回了超过 **CFA 7,610亿** 的未缴税款
- 2023年，过渡期总统 Assimi Goïta 签署新矿业法

### 关键变化 Key Changes (vs 旧法)

| 条款 | 旧法 (2019前) | 新法 (2023) | PMH影响 |
|---|---|---|---|
| **政府免费持股** Free Carried Interest | 10% | **最低10%** (可协商更高) | 你的股权被稀释 |
| **政府可选增持** Optional Participation | 10% | **20%** | 进一步稀释 |
| **本地私人持股** Local Private | 0% | **5%** | 需让渡给马里人 |
| **外资最大持股** Foreign Max | 80% | **65%** | 显著降低 |
| **权利金 Royalty** | 3-6.5% | **10%** | 大幅增加 |
| **企业税优惠期** Tax Holiday | 15年 | **3年** | 几乎取消 |
| **暴利税 Windfall Tax** | 无 | **新增** | 高金价时额外税负 |
| **财政稳定期** Fiscal Stability | 项目全生命周期 | **10年上限** | 长期风险增加 |

### 通俗解释 Simple Explanation

> 打个比方: 如果你在马里发现了一个金矿，想要开发:
> - **老法律**: 你投入所有资金和风险，保留80%的利润。政府拿20%但不出钱。
> - **新法律**: 你投入所有资金和风险，但最多只能保留65%。政府拿35%（其中10%完全免费+20%可选增持+5%给当地人）。而且你的权利金从3%涨到10%，税收优惠从15年缩短到3年，还可能交暴利税。

### 适用范围 Applicability

**重要区分:**
- **新矿权/续期**: 完全适用新法
- **已有矿权**: 旧法仍适用（但续期时按新法）
- **本地化法**: 对**所有现有项目**立即生效（无过渡期）

这意味着: B2Gold的Fekola虽然在旧法下运营，但续期时必须按新法谈判。这就是为什么B2Gold 2024年要与马里政府重新谈框架协议。

## 本地化要求 Local Content Requirements

### 本地采购 Local Procurement
- 矿业公司须优先使用马里当地的供应商和服务商
- 需提交年度本地采购报告

### 本地雇佣 Local Employment
- 优先雇佣马里公民
- 关键技术岗位如果没有合格的马里人，可以雇外国人，但必须制定培训计划，逐步由本地人接替

### 矿业发展基金 Mining Development Fund
- 政府从权利金中拿出 **20%** 用于当地发展
- 矿业公司另外贡献 **营业额的0.25%**
- 资金用于: 矿区周边社区的基础设施建设

## 实际影响案例 Real-World Impact

### B2Gold Fekola — 与马里政府的谈判

*来源: [B2Gold 2024年9月新闻稿](https://www.globenewswire.com/news-release/2024/09/11/2944881/0/en/B2Gold-Reaches-Agreement-on-Terms-with-Mali-Government-Relating-to-the-Framework-for-the-Fekola-Complex-Approvals-for-Fekola-Regional-and-Fekola-Underground-to-be-Expedited.html)*

2024年9月B2Gold与马里政府达成协议:
- 解决了审计遗留问题 (税务、关税)
- 同意对燃油进口开始缴税 (此前免税)
- 作为交换，获得了**2%的收入税和权利金减免** (基本抵消燃油税增加)
- 明确了2023年新矿业法对Fekola Complex的适用方式
- 获得了地下开采和Fekola Regional新许可的快速审批

**教训**: 即使在政策收紧的环境下，如果公司愿意谈判让步，仍然可以找到双方都能接受的方案。

### Resolute Mining — 更激烈的冲突
- Resolute Mining的Syama矿也面临马里政府的审计和税务追缴
- 管理层被拘留
- 提醒: 在马里做生意需要极其谨慎的政府关系管理

## PMH在马里的策略建议

1. **充分了解法律**: 在做任何投资决策前，必须请专业矿业律师审查新法
2. **预留股权空间**: 项目规划时就考虑35%的政府/本地持股
3. **建立政府关系**: 从勘探阶段就开始与矿业部、地方政府沟通
4. **本地化优先**: 尽早建立本地采购和雇佣体系
5. **合规透明**: 主动配合审计，不要试图规避
6. **B2Gold模式**: 学习B2Gold如何在困难环境中维持运营

## 矿业对马里经济的贡献 Mining's Contribution to Mali's Economy

- 2023年矿业贡献了 **CFA 6,440亿** (约$10亿) 到国家预算
- 占政府总收入的 **21.5%**
- 黄金出口是马里最大的外汇来源
- 矿业直接雇佣约 **12,000人** (间接影响数十万人)

> **关键认知**: 马里政府虽然收紧了政策，但他们非常依赖矿业收入。这意味着双方都有谈判的动力——政府需要外国投资和技术，外国公司需要资源和许可。关键是找到**双赢的平衡点**。
