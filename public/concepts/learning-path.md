# 矿业知识学习路径 Mining Knowledge Learning Path

*为商业背景、矿业零基础的学习者设计*

## 你的学习目标 Your Learning Goals

作为一个想进入矿业的商业人士，你需要掌握的不是成为地质师或工程师，而是：
1. **能看懂**技术报告和矿企年报
2. **能提出**正确的问题（比如"这个资源量估算用了什么方法？"）
3. **能判断**一个项目值不值得投资
4. **能沟通**和地质师、工程师有效对话

## 学习方法建议 Study Tips

### 每日学习节奏 Daily Rhythm
```
📖 20分钟: 阅读一个概念章节
📝 10分钟: 在笔记中记录3个关键要点
❓ 15分钟: 做对应章节的练习题
🔄  5分钟: 复习错题本中的到期题目
```
每天50分钟，5周可以完成全部内容。

### 记忆技巧 Memory Techniques
1. **类比法**: 每个概念都有通俗类比（文中已标注），先理解类比再学术语
2. **中英文对照**: 关键术语要同时记住中英文，因为行业沟通多用英文
3. **间隔复习**: 系统会自动安排错题复习，按遗忘曲线巩固
4. **实战关联**: 学到的知识尝试用在看公司年报上（推荐看B2Gold和Newmont的年报）
