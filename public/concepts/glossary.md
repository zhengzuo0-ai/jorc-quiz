# 矿业术语表 Mining Glossary 中英文对照

## A

| 英文 English | 中文 Chinese | 定义 Definition |
|---|---|---|
| AISC | 全维持成本 | All-In Sustaining Cost — 矿山维持运营的完整成本 |
| Alluvial | 冲积的 | 由河流沉积形成的 (常用于砂金矿) |
| Alteration | 蚀变 | 岩石被热液流体改变矿物组成 |
| AMD | 酸性矿山排水 | Acid Mine Drainage — 硫化物氧化产生的酸性水 |
| Anomaly | 异常 | 偏离背景值的地球化学或地球物理信号 |
| Arsenopyrite | 毒砂 | 含砷的铁硫化物 (FeAsS)，金的重要载体矿物 |
| Assay | 化验/分析 | 测定矿石中金属含量 |
| AusIMM | 澳大利亚矿冶学会 | 胜任人的认证机构之一 |

## B

| 英文 | 中文 | 定义 |
|---|---|---|
| Bench | 台阶 | 露天矿中的开采台阶 |
| Beneficiation | 选矿 | 提高矿石品位的处理过程 |
| BIF | 条带状铁建造 | Banded Iron Formation — 前寒武纪含铁沉积岩 |
| Blast hole | 炮孔 | 用于装药爆破的钻孔 |
| Block model | 块体模型 | 将矿体划分为三维块体的计算机模型 |
| Bonanza | 特高品位矿化 | 局部极富矿化 (通常 >30 g/t Au) |

## C

| 英文 | 中文 | 定义 |
|---|---|---|
| CAPEX | 资本支出 | Capital Expenditure — 建设矿山的初始投资 |
| Carlin-type | 卡林型 | 微细浸染型金矿 (以内华达州Carlin命名) |
| Chargeability | 充电率 | IP测量的关键参数，反映硫化物含量 |
| CIL | 炭浸法 | Carbon-in-Leach — 浸出和吸附同步进行 |
| CIP | 炭浆法 | Carbon-in-Pulp — 先浸出后吸附 |
| Competent Person | 胜任人 | JORC Code规定的合格报告签署人 |
| Concentrate | 精矿 | 选矿后品位提高的产品 |
| Core | 岩心 | 金刚石钻探获取的圆柱形岩石样品 |
| CRIRSCO | 国际矿储委 | 国际报告标准协调委员会 |
| Cut-off grade | 截断品位 | 区分矿石和废石的最低品位 |
| Cyanidation | 氰化 | 用氰化钠溶液溶解金的方法 |

## D

| 英文 | 中文 | 定义 |
|---|---|---|
| DDH | 金刚石岩心钻 | Diamond Drill Hole — 获取连续岩心的钻探方法 |
| Decline | 斜坡道 | 地下矿山从地表通往矿体的倾斜通道 |
| Dilution | 贫化 | 开采过程中混入的废石，降低品位 |
| Doré | 粗金锭 | 含金85-95%的初级冶炼产品 |
| DCF | 折现现金流 | Discounted Cash Flow — 矿业项目估值的标准方法 |
| Due diligence | 尽职调查 | 收购前的全面审查 |
| Dump leach | 堆浸 | 将矿石堆放后用化学溶液浸提金 |

## E-F

| 英文 | 中文 | 定义 |
|---|---|---|
| Epithermal | 浅成低温热液 | 与火山活动相关的浅部金矿化类型 |
| Elution | 解吸 | 从载金活性炭中洗出金 |
| Electrowinning | 电积 | 通过电解从溶液中沉积金 |
| Exploration | 勘探 | 寻找矿床的系统工作 |
| Feasibility Study | 可行性研究 | 确定项目技术和经济可行性的详细研究 |
| Fire Assay | 火试金 | 最权威的金分析方法 |
| Flotation | 浮选 | 利用矿物表面性质差异的选矿方法 |
| FPIC | 自由事先知情同意 | Free Prior and Informed Consent |

## G-I

| 英文 | 中文 | 定义 |
|---|---|---|
| g/t | 克/吨 | 品位单位 = ppm |
| Geochemistry | 地球化学 | 研究地球化学元素分布和迁移 |
| Geophysics | 地球物理 | 利用物理性质差异探测地下结构 |
| Gossanous | 铁帽状的 | 硫化物矿体地表氧化后的残留物 |
| Grade | 品位 | 矿石中有用组分的含量 |
| Greenstone belt | 绿岩带 | 太古代变质火山-沉积岩系列 |
| Heap leach | 堆浸 | 低品位矿石的浸出方法 |
| Hydrothermal | 热液的 | 与热水溶液相关的地质过程 |
| Indicated | 标示的 | JORC资源量分类的中间级别 |
| Inferred | 推断的 | JORC资源量分类的最低级别 |
| IP | 激发极化 | Induced Polarization — 检测硫化物的物探方法 |
| IRR | 内部收益率 | Internal Rate of Return |

## N-P

| 英文 | 中文 | 定义 |
|---|---|---|
| NAV | 净资产价值 | Net Asset Value |
| NPV | 净现值 | Net Present Value |
| Nugget effect | 块金效应 | 变差函数中零距离方差，反映微观不均匀性 |
