# 勘探流程 | Exploration Pipeline

> 💡 **从第一张卫星图到矿山投产，黄金勘探是一场平均耗时 7-15 年、淘汰率超过 99% 的漫长征途——理解每个阶段的投入、产出和决策关口，是矿业投资的核心能力。**

## 核心概念 (Core Concepts)

### 1. 完整勘探流程 (Full Exploration Pipeline)

#### 阶段一：桌面研究 (Desktop Study / Generative Phase)

| 参数 | 详情 |
|------|------|
| **时间** | 2-4 周 |
| **预算** | $10,000 - $50,000 |
| **目标** | 评估项目的初步潜力，决定是否值得实地考察 |

**主要工作内容**：
- **文献综述 (Literature Review)**: 收集目标区域的地质报告、学术论文、政府地质调查数据、前期勘探公司的历史报告
- **遥感分析 (Remote Sensing Analysis)**: 利用卫星影像（ASTER, Landsat, Sentinel-2）识别蚀变带、构造线性体和岩性单元
- **数据编录 (Data Compilation)**: 将所有可用数据整合到 GIS（地理信息系统）中，建立项目数据库
- **许可证尽调 (Tenure Due Diligence)**: 确认矿权有效性、面积、到期时间、年度义务支出
- **初步靶区圈定 (Preliminary Target Generation)**: 综合所有信息，初步识别优先勘探区域

**产出物**: 项目评估报告、GIS 数据库、初步靶区图、Go/No-Go 建议

#### 阶段二：现场踏勘 (Reconnaissance / Field Visit)

| 参数 | 详情 |
|------|------|
| **时间** | 2-4 周 |
| **预算** | $50,000 - $200,000 |
| **目标** | 地面验证桌面研究结论，获取第一手地质信息 |

**主要工作内容**：
- **地表地质填图 (Surface Geological Mapping)**: 观察露头 (Outcrops)、记录岩性和构造，验证遥感解译
- **岩石采样 (Rock Chip Sampling)**: 从露头直接取样进行金及多元素分析
- **土壤采样 (Reconnaissance Soil Sampling)**: 宽间距（200-500m）的土壤样品，进行初步地球化学普查
- **ASM 调查**: 在西非，记录手工采矿活动的位置、规模和矿化类型
- **社区与物流评估**: 了解通达性、水电供应、社区态度和安全状况

**产出物**: 现场报告、采样分析结果、靶区优先级更新

#### 阶段三：系统勘探 (Systematic Exploration)

| 参数 | 详情 |
|------|------|
| **时间** | 6-18 个月 |
| **预算** | $500,000 - $3,000,000 |
| **目标** | 确认金矿化的存在和范围，精确定义钻探靶区 |

**主要工作内容**：
- **网格化土壤采样 (Grid Soil Sampling)**: 间距 25-200m，系统覆盖优先区域，检测金及指示元素（As, Sb, W, Bi 等路径元素 Pathfinder Elements）
- **地球物理测量 (Geophysical Surveys)**: 机载磁法 (Airborne Magnetics)、地面激发极化法 (Ground IP)、电磁法 (EM) 等
- **初步钻探**: 螺旋钻 (Auger) 或旋转气吹钻 (RAB, Rotary Air Blast) 穿透风化壳取样——这是西非勘探中极常用的低成本钻探手段
- **详细地质填图 (Detailed Geological Mapping)**: 1:10,000 - 1:25,000 比例尺

**产出物**: 地球化学异常图、物探解译图、初步钻探截面图、钻探靶区建议书

**关键决策点**: 是否存在具有经济意义的金异常？异常的规模和强度是否值得进入钻探阶段？

#### 阶段四：靶区钻探 (Target Drilling)

| 参数 | 详情 |
|------|------|
| **时间** | 12-24 个月 |
| **预算** | $2,000,000 - $10,000,000 |
| **目标** | 验证靶区矿化，初步圈定矿体，估算推断资源量 (Inferred Resources) |

**主要工作内容**：
- **反循环钻探 (RC Drilling, Reverse Circulation)**: 成本约 $80-150/m，速度快，适合初步验证和资源量定界。西非最常用的金矿勘探钻探方法。
- **金刚石钻探 (Diamond Drilling / Core Drilling)**: 成本约 $150-300/m，获取完整岩芯 (Core)，用于详细地质研究、矿化学和冶金测试
- **钻探间距**: 初期通常 100-200m 间距，局部加密至 50-100m
- **采样与化验 (Sampling & Assaying)**: 系统切割/劈样、送实验室进行多元素分析

**产出物**: 钻探数据库、矿化截面图和平面图、初步三维地质模型、**首次资源量估算 (Maiden Resource Estimate)**——通常以推断级别 (Inferred) 为主

**重要里程碑**: 首次资源量公告是勘探公司的关键价值转折点，通常伴随股价大幅波动。

#### 阶段五：加密钻探 (Infill Drilling)

| 参数 | 详情 |
|------|------|
| **时间** | 12-36 个月 |
| **预算** | $5,000,000 - $20,000,000 |
| **目标** | 提升资源级别至控制 (Indicated) 和探明 (Measured)，支持可行性研究 |

**主要工作内容**：
- **加密 RC 和钻石钻**: 钻探间距加密至 25-50m（取决于矿化类型和连续性）
- **地质建模 (Geological Modeling)**: 建立详细的三维矿体模型和品位模型
- **矿石选冶试验 (Metallurgical Testwork)**: 从钻孔岩芯中取代表性样品进行选矿和冶金试验，确定最佳工艺流程和金回收率
- **环境基线调查 (Environmental Baseline Studies)**: 开始收集环境本底数据（水质、生态、社会经济），为后续环境影响评估 (EIA) 做准备
- **岩土工程调查 (Geotechnical Studies)**: 评估边坡稳定性（露天矿）或围岩条件（地下矿）

**产出物**: 更新的资源量估算（Indicated + Measured 级别大幅提升）、选冶报告、环境基线报告

#### 阶段六：可行性研究 (Feasibility Studies)

可行性研究分三个递进层级：

**概略研究 (Scoping Study / Preliminary Economic Assessment, PEA)**：
| 参数 | 详情 |
|------|------|
| **时间** | 3-6 个月 |
| **预算** | $200,000 - $1,000,000 |
| **精度** | ±35-50% |
| **目的** | 初步评估项目经济可行性，决定是否值得进一步投入 |

**预可行性研究 (Pre-Feasibility Study, PFS)**：
| 参数 | 详情 |
|------|------|
| **时间** | 6-12 个月 |
| **预算** | $2,000,000 - $5,000,000 |
| **精度** | ±25-30% |
| **目的** | 确定项目的技术和经济可行性，选定优先开发方案 |

**最终可行性研究 (Definitive Feasibility Study, DFS / Bankable Feasibility Study, BFS)**：
| 参数 | 详情 |
|------|------|
| **时间** | 12-24 个月 |
| **预算** | $5,000,000 - $30,000,000 |
| **精度** | ±10-15% |
| **目的** | 为银行融资 (Project Finance) 提供依据，确定最终设计参数 |

**可行性研究核心内容**：
- 矿产储量 (Ore Reserves) 估算（从资源量转化，应用修正因素）
- 采矿设计和生产计划 (Mine Plan)
- 选矿和冶金流程设计 (Process Design)
- 基础设施规划（道路、供电、供水、尾矿库）
- 资本支出 (CAPEX) 和运营支出 (OPEX) 估算
- 财务模型：NPV (净现值)、IRR (内部收益率)、回收期 (Payback Period)
- 环境和社会影响评估 (ESIA)
- 风险评估和敏感性分析

#### 阶段七：建设与投产 (Construction & Commissioning)

| 参数 | 详情 |
|------|------|
| **时间** | 18-36 个月 |
| **预算** | $50,000,000 - $500,000,000+（取决于项目规模） |
| **目标** | 建成矿山并达到设计产能 |

**主要工作内容**：
- 详细工程设计 (Detailed Engineering Design)
- 采购设备和材料 (Procurement)
- 土建施工 (Civil Construction): 选矿厂、道路、营地、尾矿设施
- 设备安装和调试 (Installation & Commissioning)
- 招聘和培训运营团队
- 产量爬坡 (Ramp-up) 至设计产能

### 2. 决策关口体系 (Stage-Gate Decision Framework)

每个阶段之间都有一个**决策关口 (Decision Gate / Go-No-Go Point)**——这是管理层/董事会根据阶段成果决定"继续投入"、"暂停"还是"退出"的关键时刻。

```
桌面研究 → [Gate 1] → 现场踏勘 → [Gate 2] → 系统勘探 → [Gate 3] → 靶区钻探 → [Gate 4] → 加密钻探 → [Gate 5] → 可行性研究 → [Gate 6] → 建设投产
```

**Gate 的核心问题**：
- Gate 1: 桌面数据是否支持实地考察？
- Gate 2: 地表证据是否支持系统勘探投资？
- Gate 3: 地球化学和物探异常是否值得钻探？
- Gate 4: 首次资源量是否满足最低规模/品位门槛？
- Gate 5: 资源量级别和选冶结果是否支持可行性研究？
- Gate 6: DFS 结果是否满足投资回报要求（通常 IRR > 20%）？

### 3. 成功率统计 (Success Rate Statistics)

勘探是一项高风险活动，统计数据令人清醒：

```
1,000 个勘探项目 (Grassroots)
    ↓ 10% 通过率
  100 个进入钻探
    ↓ 10% 通过率
   10 个达到资源量估算
    ↓ 10-30% 通过率
  1-3 个最终成为矿山
```

- **总体成功率**: 约 0.1-0.3%（每 300-1,000 个草根项目产生 1 个矿山）
- **从发现到生产**: 通常 **7-15 年**，部分复杂项目超过 20 年
- **总投资**: 从勘探到投产的累计投资通常为 $100M-$1B+

### 4. 现金流 J 曲线 (Cash Flow J-Curve)

矿业项目的现金流呈现典型的 **J 曲线**特征：

```
现金流
  ^
  |            投产
  |             |     /‾‾‾‾‾‾‾‾‾‾
  |             |    /
  |             |   /
  |-----------+----|--→ 时间
  |          /|
  |         / |  建设期
  |        /  |（最大资金需求）
  |  勘探 /
  |______/
```

- **勘探期 (3-10年)**: 持续现金流出，金额相对较小但累积可观
- **建设期 (2-3年)**: 现金流出达到峰值（CAPEX 集中投入）
- **产量爬坡期 (6-12月)**: 开始有收入但尚未达到满产
- **稳态生产期 (10-20年)**: 正现金流，逐步回收投资
- **回收期 (Payback Period)**: 通常从投产后 3-7 年

**对 PMH 的资金规划启示**: 勘探阶段可以用较小资金推进（每年 $1-5M），但一旦进入建设阶段，资金需求将急剧上升，需要提前规划融资策略。

### 5. 阶段融资策略 (Stage-Gate Financing)

不同勘探阶段吸引不同类型的投资者：

| 阶段 | 典型投资者 | 融资方式 |
|------|----------|---------|
| 桌面研究/踏勘 | 创始人、天使投资者 | 自有资金、种子轮 |
| 系统勘探 | 风险资本 (Venture Capital)、早期矿业基金 | 私募股权、IPO |
| 靶区钻探 | 矿业专注基金、战略投资者 | 定向增发 (Placement)、JV 合资 |
| 加密钻探/PFS | 中型矿业公司 (Mid-tier)、资源基金 | 股权融资、流媒体/特许权 (Stream/Royalty) |
| DFS/建设 | 银行、大型矿业公司 (Major)、DFI | 项目融资 (Project Finance)、债务融资 |
| 投产运营 | 商业银行、资本市场 | 循环信贷 (Revolving Credit)、债券 |

**JV (合资企业)** 模式在勘探阶段尤为常见：小型勘探公司发现靶区后，引入资金实力更强的合作伙伴 (通常是 Mid-tier 或 Major)，由后者提供勘探资金以赚取项目权益 (Earn-in)。

## PMH 实践关联 (PMH Context)

> PMH 作为一家在西非和纳米比亚运营的矿业公司，勘探流程管理是其核心竞争力之一：
>
> - **当前阶段定位**: PMH 需要清晰地向投资者传达每个项目处于哪个勘探阶段，以及进入下一阶段的条件和时间表。这直接影响公司估值。
> - **西非勘探成本特点**: 在西非，由于深厚风化壳，系统勘探阶段可能需要更多的 Auger/RAB 钻探来穿透表层红土，预算需相应调整（通常比在露头良好的地区多 30-50%）。
> - **成功率管理**: PMH 应建立项目组合 (Portfolio) 管理思维——不把所有资金投入单一项目，而是同时推进多个不同阶段的项目，通过 Stage-Gate 体系持续筛选和优化组合。
> - **融资时机**: 在金价高位期间（如当前 $2,300+/oz），矿业融资环境较好。PMH 应利用有利窗口完成关键阶段的融资，为 J 曲线底部（建设期）储备资金。
> - **JV 策略**: 对于资金有限的 PMH，JV 模式值得重点考虑——用发现的靶区吸引大型矿业公司的 Earn-in 投资，既能推进项目又能降低稀释。
