# 金矿开采方法 Gold Mining Methods

*参考资料: [Wikipedia - Heap Leaching](https://en.wikipedia.org/wiki/Heap_leaching), [Mining Pedia - Heap Leach Guide](https://www.miningpedia.cn/cyanidation/complete-guide-for-gold-heap-leaching-process.html), [Farmonaut - Heap Leaching 2025](https://farmonaut.com/mining/heap-leaching-for-gold-extraction-7-key-advantages-2025)*

> **通俗理解**: 开采金矿就像挖宝——首先要找到"宝箱"（矿体），然后选择最经济的方式把它挖出来。如果宝箱埋得浅，就从上面挖个大坑（露天开采）；如果埋得深，就打一条隧道进去（地下开采）；如果金子分布得很散、品位很低，那就把矿石堆起来慢慢"泡"出金子（堆浸）。

## 开采方法分类 Classification

| 方法 Method | 适用条件 | 品位要求 | 产能 |
|---|---|---|---|
| 露天开采 Open-pit | 浅表、大规模 | 0.5-2 g/t | 高 High |
| 地下开采 Underground | 深部、高品位 | 3-10+ g/t | 中 Medium |
| 堆浸 Heap Leach | 低品位氧化矿 | 0.3-1.0 g/t | 低-中 |
| 砂金开采 Placer Mining | 冲积砂金 | 可变 | 低 |

## 露天开采 Open-Pit Mining

### 基本原理 Basic Principle
- 从地表向下逐层开采，形成**台阶 (benches)**
- 剥离废石 (waste rock) 暴露矿体
- **剥采比 Stripping Ratio (SR)**: 废石量 : 矿石量
  - 典型值: 2:1 到 8:1
  - SR越高，开采成本越大

### 开采流程 Mining Process
1. **穿孔 Drilling** — 钻凿炮孔
2. **爆破 Blasting** — 装药爆破碎岩
3. **装载 Loading** — 电铲/液压挖掘机装载
4. **运输 Hauling** — 矿用卡车运输 (100-400吨级)
5. **排废 Waste Dumping** — 废石堆放排土场

### 关键参数
| 参数 Parameter | 典型值 |
|---|---|
| 台阶高度 Bench Height | 10-15m |
| 坡面角 Face Angle | 70-80° |
| 整体坡角 Overall Slope Angle | 40-55° |
| 炮孔直径 Blast Hole Diameter | 150-310mm |
| 最小工作宽度 Minimum Working Width | 30-50m |

### 代表性矿山
- **Super Pit, Kalgoorlie (澳大利亚)** — 3.5km × 1.5km × 600m深
- **Grasberg (印尼)** — 世界最大金矿
- **Muruntau (乌兹别克斯坦)** — 世界最大露天金矿

## 地下开采 Underground Mining

### 常用方法 Common Methods

#### 充填采矿法 Cut-and-Fill
- 自下而上逐层开采
- 每层采完后回填 (paste fill / hydraulic fill)
- 适合: 高品位、不规则矿体
- 矿石回收率高 (>90%)

#### 分段崩落法 Sublevel Caving
- 矿体从顶部开始崩落
- 利用重力使矿石向下运移
- 适合: 厚大陡倾矿体
- 矿石损失率较高

#### 房柱法 Room-and-Pillar
- 留下矿柱支撑顶板
- 适合: 水平/缓倾薄矿体
- 回收率: 50-75% (矿柱中的矿石损失)

#### 长壁法 Longwall
- 较少用于金矿
- 主要用于南非深部金矿 (如Witwatersrand)
- 深度可达 **3,000m+**

### 地下开采的挑战
- **通风 Ventilation**: 深部温度高 (地温梯度 25-30°C/km)
- **地压 Ground Pressure**: 深部岩爆 (rockburst) 风险
- **排水 Dewatering**: 地下水管理
- **成本**: 地下开采成本通常是露天的 3-5 倍

## 堆浸 Heap Leaching

> **通俗理解**: 堆浸就像"泡咖啡"——把咖啡粉（碎矿石）堆在一个过滤纸（防渗膜）上，然后慢慢淋上热水（氰化钠溶液），咖啡的味道（金）会溶入水中，从底部流出来被收集。只不过这"咖啡"要泡60-120天。

### 原理 Principle
将低品位矿石堆放在防渗垫层上，喷淋稀氰化钠溶液，金被溶解后从底部收集含金贵液。

### 流程图 Process Diagram

```
  💧 氰化钠溶液 (喷淋/滴灌)
        ⬇️ ⬇️ ⬇️ ⬇️ ⬇️
   ┌─────────────────────────┐
   │   碎矿石堆 ORE HEAP     │  ← 金随溶液向下溶解
   │   (高5-10米)             │     Gold dissolves as
   │                         │     solution percolates
   └─────────────────────────┘
   ═══════════════════════════  ← 防渗膜 HDPE Liner
        ⬇️ ⬇️ ⬇️ ⬇️ ⬇️
   ┌─────────────────────────┐
   │  含金贵液 (PLS)          │  ← 在此收集
   │  Pregnant Leach Solution │
   └───────────┬─────────────┘
               ⬇️
   ┌─────────────────────────┐
   │  加工厂 Processing       │  ← 活性炭吸附 或
   │  (Carbon / Zinc)        │     锌粉置换回收金
   └───────────┬─────────────┘
        ⬇️              ⬇️
   纯金 Gold         贫液循环利用 🔄
   (熔炼)           Barren solution recycled
```

### 关键参数
- 氰化钠浓度: 0.3-1.0 kg/t NaCN
- 喷淋速率: 5-10 L/h/m²
- 浸出周期: **60-120 天** (氧化矿石较快，硫化矿石更慢)
- 金回收率: **60-80%** (低于CIL/CIP的90%+，但成本低得多)
- 可处理品位低至 **0.2 g/t** 的矿石
- 比传统选矿厂节能 **40-55%**，节水约 **30%**

### 优势与局限
| 优势 Advantages | 局限 Limitations |
|---|---|
| 投资成本低 | 回收率较低 |
| 适合低品位矿 | 受气候影响大 |
| 运营简单 | 环保要求严格 |
| 可处理大量矿石 | 不适合含泥矿石 |

## 砂金开采 Placer Mining

### 方法类型
1. **淘金盘 Gold Panning** — 最原始方法，利用密度差分离
2. **溜槽 Sluice Box** — 水流+重力，连续回收
3. **摇床 Rocker Box** — 小规模，手动操作
4. **采金船 Dredging** — 水上作业，处理河底砂砾
5. **液压采矿 Hydraulic Mining** — 高压水枪冲刷

---

*关键对比: 露天=大规模低成本、地下=高品位高成本、堆浸=低品位低投资、砂金=传统简单*
