# 技术研究 Technical Studies

## 三级技术研究体系 Three-Level Study Framework

### 概略研究 Scoping Study (Order of Magnitude)
- **精度**: ±50%
- **目的**: 项目是否值得继续投入?
- **基于**: 推断+标示资源量 (Inferred + Indicated)
- **内容**:
  - 概念性采矿方案
  - 初步冶金测试结果
  - 粗略资本和运营成本估算
  - 简单经济分析 (NPV/IRR)
- **周期**: 2-4个月
- **成本**: $50K-$200K

### 预可行性研究 Pre-Feasibility Study (PFS)
- **精度**: ±25%
- **目的**: 项目在技术和经济上是否可行?
- **基于**: 标示+探明资源量 (Indicated + Measured)
- **内容**:
  - 初步矿山设计 (open pit / underground)
  - 详细冶金测试
  - 初步环境影响评估
  - 基于供应商报价的成本估算
  - 详细经济分析 + 敏感性分析
  - 初步储量估算
- **周期**: 6-12个月
- **成本**: $1M-$5M

### 可行性研究 Feasibility Study (DFS/BFS)
- **精度**: ±10-15%
- **目的**: 项目能否融资和建设?
- **基于**: 探明+标示资源量 → 证实+概略储量
- **内容**:
  - 详细矿山设计 (生产计划、设备选型)
  - 完整工艺流程设计
  - 详细环境影响评估 (EIS)
  - 基于工程报价的资本成本
  - 运营成本详细建模
  - 财务模型 + 融资方案
  - 项目执行计划
  - 风险评估
- **周期**: 12-24个月
- **成本**: $5M-$30M

### 各阶段对比 Comparison

| 特征 | Scoping | PFS | DFS |
|---|---|---|---|
| 精度 | ±50% | ±25% | ±10-15% |
| 资源基础 | Inferred可用 | Indicated为主 | Measured+Indicated |
| 储量 | 不估算 | 初步估算 | 正式估算 |
| 环评 | 概念性 | 初步 | 完整EIS |
| 设计深度 | 概念 | 初步工程 | 详细工程 |
| 能否融资 | 否 | 困难 | 是 |
| 决策 | 继续/停止 | 继续/修改/停止 | 建设/不建设 |

## JORC Code 对技术研究的要求

### Clause 38: Production Targets
- 基于推断资源量的产量目标必须标注**免责声明 (cautionary statement)**
- 声明内容: "推断资源量的地质确信度不足以将修正因素应用于其上来转换为储量"

### Clause 39: 预测性财务信息
- 基于Inferred的经济分析必须包含免责声明
- 不得将Inferred资源量用于矿山寿命的主要经济分析
- PFS/DFS中Inferred部分只能作为"潜在上行"展示

---

*关键记忆: Scoping=值不值得?, PFS=行不行得通?, DFS=能不能建? 精度和投入逐级递增。*
