# 金矿床类型 Gold Deposit Types

## 造山型金矿 Orogenic Gold Deposits

全球最重要的金矿类型，贡献了历史上约 **75%** 的黄金产量。

### 形成机制 Formation Mechanism
- 形成于**造山运动 (orogenesis)** 期间
- 变质流体沿断裂构造运移，在有利部位沉淀成矿
- Metamorphic fluids migrate along fault structures and precipitate gold at favorable sites
- 典型温度: **250-400°C**，压力: 1-3 kbar
- 成矿流体以 **CO₂-H₂O-NaCl** 为主

### 典型特征 Typical Characteristics
| 特征 Feature | 描述 Description |
|---|---|
| 围岩 Host Rock | 变质岩 (绿片岩相-角闪岩相) |
| 构造控制 Structural Control | 剪切带、断裂带 |
| 矿化样式 Mineralization Style | 石英脉型、蚀变岩型 |
| 伴生矿物 Associated Minerals | 黄铁矿、毒砂、方铅矿 |
| 蚀变 Alteration | 硅化、碳酸盐化、绢云母化 |
| 品位 Grade | 中-高品位 (5-30 g/t) |

### 世界著名造山型金矿
- **Witwatersrand (南非)** — 历史产金量最大
- **Kalgoorlie (澳大利亚)** — "Golden Mile"
- **Val-d'Or (加拿大)** — 阿比提比绿岩带
- **胶东半岛 (中国)** — 中国最大金矿集中区

## 浅成低温热液型 Epithermal Gold Deposits

与**火山活动 (volcanism)** 密切相关的金矿类型。

### 分类 Classification

#### 高硫化型 High-Sulfidation (HS)
- 酸性流体 (pH 1-3)
- 蚀变: 高级泥化 → 明矾石 + 高岭石
- 矿化: 以脉状/浸染状为主
- 金与铜、砷、锑伴生
- 代表矿床: **Yanacocha (秘鲁)**, **Lepanto (菲律宾)**

#### 低硫化型 Low-Sulfidation (LS)
- 近中性流体 (pH 5-7)
- 蚀变: 硅化 + 冰长石化 + 绿泥石化
- 矿化: 石英-冰长石脉
- 金与银伴生，银金比高
- 代表矿床: **Hishikari (日本)**, **Waihi (新西兰)**

### 形成条件
- 深度: **<1.5 km**（浅表环境）
- 温度: **150-300°C**
- 常见于**岛弧 (island arc)** 和**大陆边缘弧 (continental margin arc)** 环境

## 卡林型金矿 Carlin-Type Gold Deposits

以美国内华达州 **Carlin Trend** 命名的独特金矿类型。

### 核心特征
- 金呈**不可见金 (invisible gold)** 形式赋存于含砷黄铁矿中
- 围岩: 不纯碳酸盐岩、粉砂岩
- 蚀变: 去钙化 (decalcification)、硅化、硫化
- 品位: 1-10 g/t，但矿体规模大
- 全球约 **40%** 分布在内华达州

### 关键识别标志 Key Recognition Criteria
1. 沉积岩容矿 (sedimentary host rocks)
2. 微细浸染金 (disseminated submicron gold)
3. 含砷黄铁矿 (arsenian pyrite)
4. 去碳酸盐化蚀变 (decarbonation alteration)

## 砂金矿 Placer Gold Deposits

### 形成原理
- 岩金矿床经**风化剥蚀 (weathering & erosion)** 后
- 金因高密度在水流中富集沉积
- 分布于河流、海滩、古河道中

### 砂金类型
| 类型 Type | 环境 Setting | 描述 |
|---|---|---|
| 河流砂金 Alluvial | 活动河流 | 河床底部和两岸 |
| 阶地砂金 Terrace | 古河道 | 抬升的古河流沉积 |
| 残积砂金 Eluvial | 原地风化 | 矿脉附近残留 |
| 海滨砂金 Beach | 海岸线 | 波浪作用富集 |

## 斑岩型含金矿床 Porphyry Gold Deposits

- 大型低品位 (0.3-1.5 g/t Au)，但矿体规模巨大
- 常伴生铜 (**Au-Cu porphyry**)
- 与中-酸性浅成侵入岩有关
- 代表: **Grasberg (印尼)** — 世界最大金矿之一

---

*记忆技巧: 造山型 = 变质 + 构造控制 (高品位); 浅成热液 = 火山 + 浅表 (银金伴生); 卡林型 = 沉积岩 + 不可见金; 砂金 = 风化 + 重力富集*
