# 黄金冶金与精炼 Gold Metallurgy & Refining

## 金的分析方法 Gold Assay Methods

### 火试金法 Fire Assay
- **最权威、最准确**的金分析方法
- 原理: 将矿石样品与铅、硼砂等混合熔炼
- 铅吸收金银 → 灰吹 (cupellation) 去除铅 → 称重合金珠
- 用酸溶解银 → 剩余为纯金 → 称重
- 检出限: 约 0.01 g/t (10 ppb)
- **JORC Code 认可的标准方法**

### 原子吸收光谱 AAS (Atomic Absorption Spectroscopy)
- 快速、成本低
- 需要先溶解样品 (王水消解)
- 检出限: 约 0.01 g/t
- 常用于常规分析

### ICP-MS (电感耦合等离子体质谱)
- 可同时分析多种元素
- 极低检出限: ppb级
- 适合多元素地球化学分析
- 成本较高

### 对比
| 方法 | 精度 | 成本 | 速度 | 适用 |
|---|---|---|---|---|
| Fire Assay | 最高 | 高 | 慢 | 资源估算、仲裁 |
| AAS | 高 | 中 | 快 | 常规分析 |
| ICP-MS | 高 | 高 | 快 | 多元素分析 |

## Doré 金锭生产 Doré Bar Production

### 从矿石到金锭的流程

```
矿石 → 破碎磨矿 → 重选 (回收粗粒金) → 浮选 (富集硫化物)
                                              ↓
    金锭 (Doré) ← 熔炼 ← 电积 ← 解吸 ← CIL/CIP氰化浸出
```

### 电积 Electrowinning
- 含金溶液通过电积槽
- 金在阴极 (cathode) 上沉积
- 产出金泥 (gold sludge)
- 含金量: 70-90%

### 熔炼 Smelting
- 金泥 + 助熔剂 (硼砂、碳酸钠)
- 温度: 1,200-1,400°C
- 产出 Doré bar (粗金锭)
- 含金量: 85-95%

## 精炼 Refining

### Wohlwill 电解精炼法
- 最常用的高纯度金精炼方法
- Doré bar作为阳极 → 纯金沉积在阴极
- 产出 99.99% (4N) 纯金
- 回收副产品 (银、铂族金属)

### Miller 氯化法
- 用氯气通入熔融金中
- 氯化去除贱金属和银
- 产出 99.5% 纯金
- 速度快于Wohlwill，但纯度稍低

### LBMA Good Delivery标准
| 参数 | 要求 |
|---|---|
| 纯度 Fineness | ≥99.5% |
| 重量 Weight | 350-430 troy oz (约10.9-13.4 kg) |
| 标记 Marks | 精炼厂标志、序列号、纯度、年份 |
| 外观 Appearance | 无气泡、裂缝 |

---

*从矿石到金条的完整链条: 采矿→选矿→氰化→电积→熔炼→精炼→Good Delivery bar*
