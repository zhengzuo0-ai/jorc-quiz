# 勘探结果报告 Reporting of Exploration Results

## 概述 Overview

勘探结果 (Exploration Results) 是JORC Code中尚未构成矿产资源量估算的地质和分析数据。

## 报告原则 Reporting Principles

### 什么必须报告 What Must Be Reported

JORC Code Table 1 (Section 1) 要求披露:

| 项目 Item | 要求 Requirement |
|---|---|
| 取样技术 Sampling Techniques | 钻探类型、取样方法、样品长度 |
| 样品记录 Logging | 地质编录方法和详细程度 |
| 钻孔信息 Drill Hole Info | 钻孔编号、方位、倾角、深度 |
| 数据汇总方法 Data Aggregation | 品位加权方法、截断值 |
| 金属等价物 Metal Equivalents | 如使用，须说明假设 |
| 与矿化带的关系 Relationship to Mineralization | 真厚度vs视厚度 |
| 图件 Diagrams | 必须包含足够的图件辅助理解 |
| 平衡报告 Balanced Reporting | 不能只报告"好消息" |

### 关键要求详解

#### 真厚度 vs 视厚度 True Width vs Apparent Width

```
视交叉长度 (Intersection length) ≠ 真厚度 (True width)

真厚度 = 视交叉长度 × sin(α)
其中 α = 钻孔与矿体的交角
```

**JORC要求**: 如果报告交叉口长度，必须说明:
- 这是视交叉长度还是真厚度
- 如果真厚度不明确，必须声明

#### 截断品位 Cut-off Grade

报告勘探结果时必须说明:
- 使用了什么截断品位 (如 0.5 g/t Au)
- 最大内部稀释宽度 (如 2m)
- 最小矿化宽度 (如 1m)

#### 异常值处理 High-Grade Handling

- 如果交叉口包含个别高品位样品:
  - 必须单独标注高品位间距
  - 报告有/无高品位时的加权平均品位
  - 例如: "12m @ 5.2 g/t Au (包含 1m @ 45 g/t)"

### 禁止事项 What NOT to Do

| 错误做法 | 正确做法 |
|---|---|
| 只报告最好的钻孔结果 | 报告所有钻孔，包括贫矿钻孔 |
| 报告"品位高达 X g/t" | 报告加权平均品位和截断条件 |
| 使用"矿体" "矿石" 等词 | 在资源估算前使用"矿化" "异常"等词 |
| 隐瞒不利结果 | 全面、平衡地报告 |
| 暗示经济可行性 | 明确声明尚未进行经济评估 |

## QA/QC 质量控制 Quality Assurance/Quality Control

### 标准参考样 Certified Reference Materials (CRM)
- 在样品批次中插入已知品位的标准样
- 监控实验室分析准确度
- 如果CRM结果偏离>2σ → 该批次需要复查

### 空白样 Blank Samples
- 在样品批次中插入已知无矿化的样品
- 检测交叉污染
- 特别重要于高品位金矿采样

### 重复样 Duplicate Samples
- **现场重复**: 同一采样点取两个样品
- **粗副样重复**: 粗碎后分出副样送不同实验室
- **实验室内重复**: 实验室内部重复分析

### QA/QC 插入频率

| 类型 | 建议频率 | 目的 |
|---|---|---|
| CRM | 每20-25个样品1个 | 检查准确度 |
| 空白样 | 每20-30个样品1个 | 检查污染 |
| 现场重复 | 每20个样品1个 | 检查取样精度 |
| 粗副样 | 每20个样品1个 | 检查实验室间偏差 |

---

*考试重点: 勘探结果报告必须做到"平衡、完整、不误导"，QA/QC是数据质量的保证，真厚度与视厚度必须区分。*
