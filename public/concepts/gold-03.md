# 金矿勘探方法 Gold Exploration Methods

## 勘探阶段概述 Exploration Stages

| 阶段 Stage | 目的 Purpose | 方法 Methods | 成本 Cost |
|---|---|---|---|
| 区域踏勘 Reconnaissance | 缩小靶区 | 遥感、区域地化 | 低 Low |
| 普查 Prospecting | 发现异常 | 地质填图、土壤地化 | 中低 |
| 详查 Detailed Exploration | 查明矿体 | 钻探、物探 | 高 High |
| 勘探 Resource Delineation | 圈定资源 | 系统钻探、取样 | 很高 Very High |

## 地质方法 Geological Methods

### 地质填图 Geological Mapping
- **基础工作**: 确定岩性、构造、蚀变分布
- 比例尺: 区域 1:50,000 → 矿区 1:2,000
- 重点关注:
  - 断裂构造 (Fault structures)
  - 蚀变带 (Alteration zones)
  - 石英脉系统 (Quartz vein systems)
  - 围岩类型 (Host rock lithology)

### 构造分析 Structural Analysis
- 金矿化常受构造控制 (structurally controlled)
- 关键构造类型:
  - **剪切带 Shear zones** — 造山型金矿的主要控矿构造
  - **断裂交汇处 Fault intersections** — 有利的金富集部位
  - **褶皱转折端 Fold hinges** — 应力集中区
  - **层间滑动面 Bedding-plane faults** — 卡林型金矿常见

## 地球化学方法 Geochemical Methods

### 土壤地球化学 Soil Geochemistry
最常用的金矿勘探方法之一。

**原理 Principle:**
- 矿体在地表形成地球化学异常 (geochemical anomaly)
- 通过系统采集土壤样品，分析元素含量
- 圈出异常区域作为钻探靶区

**采样方法 Sampling:**
- 网格采样 Grid sampling: 通常 50m × 200m 或 25m × 100m
- B层土壤 (B-horizon): 最具代表性
- 分析元素: Au, As, Sb, Hg, Cu, Pb, Zn, Ag

**指示元素 Pathfinder Elements:**
| 元素 Element | 意义 Significance |
|---|---|
| As (砷) | 金矿化最重要的指示元素 |
| Sb (锑) | 浅成热液型金矿指示 |
| Hg (汞) | 浅表矿化指示 |
| Cu (铜) | 斑岩型含金矿床 |
| Bi (铋) | 高温矿化指示 |

### 水系沉积物地球化学 Stream Sediment Geochemistry
- 用于区域踏勘阶段
- 采集河流底部沉积物
- 优点: 覆盖面广，成本低
- 可快速发现区域异常

### 岩石地球化学 Rock Geochemistry (Lithogeochemistry)
- 直接采集岩石样品分析
- 用于详查阶段
- 可准确判断矿化强度和类型

## 地球物理方法 Geophysical Methods

### 磁法 Magnetics
**原理**: 测量地球磁场的局部变化

- **航磁 Aeromagnetic survey**: 飞机搭载磁力仪，区域调查
- **地面磁法 Ground magnetics**: 精度更高
- 应用:
  - 识别磁铁矿化蚀变带 (magnetite-bearing alteration)
  - 断裂构造识别 (低磁异常)
  - BIF型金矿定位 (条带状含铁建造)

### 电法 Electrical Methods

#### 激发极化法 Induced Polarization (IP)
- **原理**: 向地下注入电流，测量充电-放电效应
- 硫化物矿物 (黄铁矿等) 产生IP效应
- 非常适合发现含硫化物的金矿化带
- "金矿最好的朋友" — IP异常常指示金矿化

#### 电阻率法 Resistivity
- 测量岩石电阻率的变化
- 石英脉 = 高阻; 蚀变带 = 低阻
- 常与IP联合使用

### 重力测量 Gravity Survey
- 测量重力场的微小变化
- 可识别:
  - 岩浆岩体 (花岗岩 = 低密度)
  - 矿体 (硫化物 = 高密度)
  - 断裂构造 (密度差异)

### 电磁法 Electromagnetic (EM) Methods
- 利用电磁感应原理
- **航空电磁 (AEM)**: 直升机/固定翼
- 适合发现:
  - 块状硫化物矿体 (massive sulfide)
  - 含水断裂带 (water-bearing faults)
  - 覆盖层厚度

## 遥感方法 Remote Sensing

### 多光谱遥感 Multispectral Remote Sensing
- **卫星**: Landsat, Sentinel-2, ASTER
- 可识别:
  - 铁染蚀变 (iron oxide alteration): 利用可见光-近红外
  - 羟基蚀变 (hydroxyl alteration): 利用短波红外 (SWIR)
  - 碳酸盐蚀变: 利用热红外 (TIR)

### 高光谱遥感 Hyperspectral Remote Sensing
- 数百个窄波段，可精确识别矿物
- 例如区分: 明矾石 vs 高岭石 vs 绢云母
- 对浅成热液型金矿蚀变填图特别有效

### 遥感在金矿勘探中的应用
| 蚀变类型 Alteration | 指示 Indication | 遥感方法 |
|---|---|---|
| 铁帽 Gossanous cap | 硫化物矿体氧化 | 可见光/NIR |
| 绢云母化 Sericitic | 热液蚀变 | SWIR |
| 硅化 Silicification | 热液通道 | TIR |
| 泥化 Argillic | 酸性蚀变 | SWIR |

## 钻探方法 Drilling Methods

### 金刚石岩心钻探 Diamond Core Drilling
- **最重要的勘探手段**
- 获取连续岩心 (core)，可进行地质编录和化验
- 岩心直径: NQ (47.6mm), HQ (63.5mm)
- 成本: $150-300/m (取决于地区)
- 优势: 提供完整地质信息

### 反循环钻探 Reverse Circulation (RC) Drilling
- 岩屑返回，非连续岩心
- 成本较低: $80-150/m
- 适合:
  - 初步评价 (grade control)
  - 浅部矿体 (<200m)
  - 大间距钻探

### 螺旋钻 Auger Drilling
- 最经济的钻探方法
- 仅用于土壤和风化层采样
- 深度通常 <30m

---

*勘探策略: 遥感(面) → 地化(线/面) → 物探(面/剖面) → 钻探(点) — 从面到点逐步缩小靶区*
*Exploration strategy: Remote sensing (area) → Geochemistry (line/area) → Geophysics (area/section) → Drilling (point)*
