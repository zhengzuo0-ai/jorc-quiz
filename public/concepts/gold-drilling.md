## 为什么钻探这么重要？ Why Drilling Matters

> **通俗理解**: 如果说物探和化探是"看病拍CT"，那钻探就是"活检取样"。前面的方法告诉你哪里可能有病灶（矿），但只有钻探能真正从地下拿出岩石来化验，确认到底有没有金子、品位多高、矿体多厚。**一个勘探项目60-80%的预算都花在钻探上**。

## 两大钻探方法 Two Main Methods

### 金刚石岩心钻探 Diamond Core Drilling (DD)

> **类比**: 想象用一个空心的圆柱形"苹果取芯器"插入苹果。拔出来后你得到了一个完整的苹果芯（岩心），可以看到苹果内部的全部结构。

**原理**: 钻头镶嵌工业金刚石，旋转切割岩石，获取连续的圆柱形岩心 (core)。

**关键参数:**
| 参数 | 规格 |
|---|---|
| 岩心直径 | NQ: 47.6mm, HQ: 63.5mm, PQ: 85mm |
| 钻进速度 | ~100 m/天 |
| 最大深度 | 2,000m+ (特殊情况可达3,400m) |
| 成本 | **$150-300/米** (取决于地区、深度、地质条件) |
| 需水量 | 较大 (冷却钻头和排渣) |

**优势 Advantages:**
- 获得完整连续岩心 — 可看到矿化结构、纹理、蚀变
- **最高的地质信息量** — 可测量、编录、拍照、做各种测试
- 孔斜控制精确
- 适合深部钻探
- **资源估算的必需** — JORC Code要求有足够的岩心钻数据

**劣势 Disadvantages:**
- 成本高 (比RC贵25-40%)
- 速度慢
- 设备复杂，搬迁耗时
- 需要大量用水

### 反循环钻探 Reverse Circulation Drilling (RC)

> **类比**: 想象用一根粗吸管插入沙堆，用高压空气从管子里把沙子吹上来。你得到的是碎沙子（岩屑），不是完整的芯样。但速度很快、成本低。

**原理**: 钻杆是双壁管 (dual tube)，压缩空气从外管向下吹，岩屑从内管向上返回。每米的岩屑通过旋风分样器 (cyclone) 收集。

**关键参数:**
| 参数 | 规格 |
|---|---|
| 孔径 | 127-140mm (5"-5.5") |
| 钻进速度 | **200-400 m/天** (比DD快2-4倍) |
| 最大深度 | 500m (特殊可达700m) |
| 成本 | **$80-150/米** (比DD便宜25-40%) |
| 需水量 | 较少 |

**优势:**
- **快速、经济** — 初期勘探的首选
- 较大的样品量 (对金矿尤其重要 — 减少块金效应)
- 全球RC钻探使用量2019-2024增长了27%
- 设备搬迁相对容易

**劣势:**
- 只获得岩屑，不是连续岩心 — 地质信息有限
- 孔斜容易偏移 ("螺旋偏斜")
- 深度受限
- 样品可能被污染 (上层岩屑混入下层)

## 对比总结 Side-by-Side Comparison

| 对比项 | RC钻探 | 金刚石岩心钻 |
|---|---|---|
| **成本** | $80-150/m | $150-300/m |
| **速度** | 200-400 m/天 | ~100 m/天 |
| **样品类型** | 岩屑 (chips) | 连续岩心 (core) |
| **地质信息** | 有限 | **最完整** |
| **最大深度** | ~500m | 2,000m+ |
| **金矿代表性** | 好 (样品量大) | 中 (半岩心取样) |
| **适用阶段** | 初步勘探、品控 | 详查、资源估算 |
| **JORC要求** | 可用于Inferred | **Indicated/Measured必需** |

## 实际应用策略 Practical Strategy

### 典型勘探钻探程序 Typical Drilling Program

```
第一阶段: RC钻探 (快速筛选)
├── 大间距网格 (100-200m)
├── 目的: 确认异常区是否有矿化
├── 预算: 较低
└── 结果: 确定有希望的区域

第二阶段: DD钻探 (详细评价)
├── 在RC异常区加密 (25-50m)
├── 目的: 获取完整地质信息
├── 预算: 较高
└── 结果: 资源量估算数据

第三阶段: RC + DD组合 (填充)
├── RC预打孔 + DD深部取芯 (同一钻孔)
├── 目的: 平衡成本和信息量
└── 结果: 完善资源量模型
```

### PMH预算建议

> **如果你有$1M钻探预算** (在马里)：
> - 方案A: 全部RC → 约8,000-10,000米 → 快速覆盖大面积
> - 方案B: 全部DD → 约4,000-5,000米 → 深度信息但覆盖面小
> - **推荐方案C**: 70% RC + 30% DD → ~6,000m RC + ~1,500m DD → 兼顾覆盖面和数据质量

## 2025年钻探新趋势 2025 Drilling Trends

1. **实时采样分析**: RC钻机配备在线分析仪，边钻边看品位
2. **数字化集成**: 钻探数据直接上传到3D资源模型
3. **自动化钻进**: 减少人工操作，提高安全性
4. **定向钻探**: 从一个钻孔位置打多个方向的钻孔 (减少搬迁)
5. **环保钻探**: 减少用水量和废弃物

---

*关键记忆: RC = 快速便宜但信息少 (像初筛)，DD = 慢贵但信息全 (像精检)。实际项目中两者结合使用。*
