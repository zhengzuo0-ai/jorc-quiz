# 矿业连续创业者 | Serial Mining Entrepreneurs

*数据来源: [Canadian Mining Hall of Fame - Ross Beaty](https://mininghalloffame.ca/ross-j-beaty/), [The Assay - Equinox](https://www.theassay.com/articles/from-equinox-to-equinox-being-an-entrepreneur-in-mining/), [WEF - Ross Beaty](https://www.weforum.org/people/ross-beaty/), [De Grey Mining](https://degreymining.com.au/), [Montage Gold](https://www.montagegold.com/)*

> 💡 **矿业中最成功的人往往不是一次性幸运儿，而是"连续创业者"——他们反复使用同一套方法论，在不同矿种和地区创造价值。理解他们的模式，就是理解矿业价值创造的核心逻辑。**

## Ross Beaty — "13家公司创造$60亿价值"

> **一句话**: 地质师+律师+环保主义者，创立13家矿业公司，每一家都盈利，总计为股东创造约$60亿价值。

**背景:**
- 1951年生于温哥华
- UBC地质学学位 → 伦敦皇家矿业学院 → UBC法学学位
- 1980年创立地质咨询公司 (30名地质师的团队)

**创业年表:**

| 年份 | 公司 | 矿种 | 结局 | 价值创造 |
|------|------|------|------|---------|
| 1985 | **Equinox Resources** | 铜 | 1994年卖给Hecla Mining | 开发了3个矿山 |
| 1994 | **Pan American Silver** | 银 | 至今运营 — 全球最大银矿公司之一 | 11座矿山, 24,000员工 |
| 2003-2010 | **Lumina Copper** | 铜 | 拆分为6家公司分别出售 | **~$20亿股东价值** |
| 2008 | Alterra Power | 地热能 | 2018年卖给Innergex | 可再生能源 |
| 2018 | **Equinox Gold** | 金 | 至今运营 — 8座矿山 | 美洲最大金矿公司之一 |
| 1994-2017 | 另外11家公司 | 各类 | 逐一出售 | 累计~$60亿 |

**Beaty的创业方程式:**

```
发现机会 → 组建团队 → 获取矿权 → 用最少的钱证明价值 → 出售或独立运营 → 重复
```

**核心哲学:**
1. **"做你懂的事"** — 坚持矿业，不搞其他行业
2. **"时机就是一切"** — 在周期低点买入（Pan American Silver成立于银价最低迷的1994年）
3. **拆分创造价值** — Lumina Copper拆成6家公司，每家独立上市，总价值远超合并状态
4. **环保不矛盾** — 全球矿业界最知名的环保主义者，每年捐$350万给72个环保组织

> **PMH教训**: Beaty证明了**矿业创业是可以"系统化"的** — 不是靠运气，而是靠方法论。他的"拆分"策略尤其值得PMH思考：如果PMH有多个项目，分别成立公司可能比打包更有价值。

## 矿业创始人的共同DNA

分析了Friedland、Bristow、Beaty、Johnson、Lassonde、Lundin等人后，他们的共同特征：

| 特征 | 说明 | PMH启示 |
|------|------|---------|
| **地质或工程背景** | 绝大多数有技术学位 | CEO不需要是地质师，但必须理解技术 |
| **极强的融资能力** | 每个人都是"融资高手" | PMH需要一个讲故事的人 |
| **逆周期思维** | 低迷时买入，高位时卖出 | 当前金价$4,000+是卖方市场 |
| **多次创业** | 不是一次性，而是反复 | 把每个项目当一家"公司"管理 |
| **全球视野** | 不局限于一个国家 | 马里+老挝+纳米比亚的组合是对的 |
| **团队连续性** | 核心团队跟随多次创业 | 找到你的"核心5人" |
| **投资纪律** | IRR>20%才投，不感情用事 | 建立Go/No-Go标准 |
| **争议不可避免** | Friedland有"Toxic Bob"，Lundin在苏丹投资 | 做好公关准备 |

## 矿业投资者的"8/80法则"

Rick Rule的经典理论，对PMH的投资组合管理极有参考价值:

```
如果你投资10个矿业项目:
- 2个会亏掉80%+ (失败)
- 6个会持平或微利 (平庸)
- 2个会赚10倍+ (成功!)

结论: 总回报由那2个成功项目决定
      → 所以关键不是避免失败
      → 而是确保你有足够多的"赌注"在桌上
      → 并且每个赌注的下注额不会让你破产
```

> **PMH应用**: 不要把所有资金投入一个项目。保持2-3个不同阶段的项目组合，接受大部分会失败，但确保有机会抓住那个"10倍回报"。

## 创始人类型矩阵 — PMH应该学谁？

| 类型 | 代表人物 | 核心能力 | 适合PMH的程度 |
|------|---------|---------|-------------|
| **探索者** | Friedland, Bristow | 发现世界级矿床 | 需要顶级地质团队 |
| **建设者** | Quartermaine (Perseus), WAF | 稳健执行，从0到生产 | **最适合PMH当前阶段** |
| **连续创业者** | Beaty, Johnson | 反复创业套用方法论 | 适合长期战略 |
| **金融家** | Lassonde, Giustra | 创新融资模式 | 适合融资策略参考 |
| **家族王朝** | Lundin, Oppenheimer | 跨代传承，长期视野 | 适合长期愿景 |
| **国有资本** | 陈景河 (紫金), 山东黄金 | 国家支持+全球收购 | 适合合作伙伴选择 |

*底线: 矿业不是博彩——最成功的人都是有方法论的连续创业者。PMH的CEO不需要成为下一个Friedland（发现天才），但可以学Quartermaine（"快速的乌龟"）或Johnson（收购被低估资产）的模式——用纪律和执行力创造价值。*
