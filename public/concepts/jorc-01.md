# JORC 三大原则 The Three Principles of JORC

*官方文件: [JORC Code 2012 Edition](https://www.jorc.org/docs/JORC_code_2012.pdf) | [ASX Listing Rules Chapter 5](https://www.asx.com.au/)*

## 概述 Overview

**JORC Code** (Joint Ore Reserves Committee Code) 是澳大利亚/亚太地区矿业公开报告的行业标准，已被纳入ASX（澳大利亚证券交易所）上市规则，因此对所有ASX上市矿业公司具有**法律约束力**。

现行版本为 **2012版**（2012年12月20日起生效），目前正在修订中，预计2025年发布新版。

JORC Code 2012 建立在三大核心原则之上，这是所有矿业公开报告的基础。

## 1. 透明度 Transparency

### 定义 Definition
> 公开报告应提供足够的信息，使读者能够理解报告内容，并不会被误导。
> Public reports should provide sufficient information to allow a reader to understand the report and not be misled.

### 关键要求
- 报告中必须包含所有**重要信息 (material information)**
- 不能选择性披露有利信息而隐藏不利信息
- 数据来源、方法和假设必须清楚说明
- 读者应能根据报告信息做出合理判断

### 实践案例
| 正确做法 | 错误做法 |
|---|---|
| 报告所有钻孔结果 | 只报告高品位钻孔 |
| 说明品位截断 (cut-off) | 不提及截断品位 |
| 披露负面结果 | 隐瞒不利钻孔 |
| 说明估算方法和参数 | 只报告最终数字 |

## 2. 重要性 Materiality

### 定义 Definition
> 公开报告应包含所有可能合理影响投资者决策的信息。
> Public reports should contain all information that investors would reasonably require to make a reasoned and balanced judgement.

### 重要性测试 Materiality Test
问自己: "这个信息如果被遗漏或错误陈述，是否会影响投资者的决策？"
- 如果是 → **必须披露**
- 如果否 → 可以省略

### 典型重要信息
- 取样方法和质量控制 (QA/QC)
- 品位截断值和估算参数
- 采矿和加工假设
- 环境和社会约束
- 经济假设 (金属价格、成本)

## 3. 胜任力 Competence

### 定义 Definition
> 公开报告必须基于由具有相关经验和专业资质的胜任人 (Competent Person) 完成的工作。
> Public reports must be based on work done by a Competent Person with relevant experience and professional qualifications.

### 胜任人资质 Competent Person Qualifications
| 要求 Requirement | 说明 |
|---|---|
| 专业会员 Professional Membership | AusIMM, AIG 或同等机构 |
| 相关经验 Relevant Experience | 该类矿床和矿化类型的 **5年以上** 经验 |
| 承担责任 Takes Responsibility | 以书面同意 (written consent) 方式确认 |
| 独立判断 Independent Judgement | 不受雇主或客户不当影响 |

### 胜任人的责任
- 确保报告准确、完整、不误导
- 亲自验证数据和解释
- 不签署自己不熟悉领域的报告
- 对公开报告中以其名义发布的信息负法律责任

## 三大原则的关系

```
透明度 (Transparency)
  ↕ 互相支撑
重要性 (Materiality)
  ↕ 互相支撑
胜任力 (Competence)
```

- **透明度**确保信息被充分披露
- **重要性**确保披露的是有意义的信息
- **胜任力**确保信息的专业质量

> **记忆口诀**: TMC — Transparent (透明), Material (重要), Competent (胜任)
> 缺少任何一个，报告就不符合JORC标准。

---

*这三个原则贯穿JORC Code的每一条款，是理解和应用JORC Code的基础。在考试中，几乎每道关于JORC原则的题目都围绕这三个概念展开。*
