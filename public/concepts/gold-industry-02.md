# 矿业估值与收购兼并 Mining Valuation & M&A

## 矿企估值方法 Valuation Methods

### 1. DCF 折现现金流法 Discounted Cash Flow

**最被银行和分析师认可的方法**

```
Enterprise Value = Σ [FCFₜ / (1 + WACC)ᵗ]

其中:
FCF = Revenue - OPEX - Tax - Capex - Working Capital Change
WACC = 矿业通常 8-12%
```

**金矿DCF的关键假设:**
| 参数 Parameter | 通常做法 | PMH参考 |
|---|---|---|
| 金价 Gold Price | 分析师共识或Forward curve | 使用3年滚动平均或当前价 |
| 矿山寿命 Mine Life | 基于储量 (P&P) | 资源量可做敏感性测试 |
| 品位 Grade | 按储量模型 | 注意稀释因子 (dilution) |
| 回收率 Recovery | 冶金试验确认 | 氧化矿 vs 硫化矿差异大 |
| AISC | 可比公司参照 | 西非通常低于全球平均 |
| 折现率 Discount Rate | WACC 8-12% | 高风险国家加2-4% |
| 税率 Tax Rate | 所在国法定税率 | 马里: 25% + 3% royalty |

### 2. NAV 净资产价值法 Net Asset Value

```
NAV = NPV of operating mines
    + Market value of development projects
    + Exploration asset value
    + Cash & investments
    - Debt
    - Environmental liabilities
```

**P/NAV 倍数:**
- 生产商 Producers: 0.8-1.2x NAV
- 开发商 Developers: 0.4-0.7x NAV
- 勘探商 Explorers: 0.2-0.5x NAV

## 尽职调查 Due Diligence

### 技术尽职调查 Technical DD
1. **资源/储量审查** — 独立CP复核估算
2. **地质模型验证** — 检查数据质量、QA/QC
3. **冶金测试** — 确认回收率
4. **采矿设计** — 验证开采计划
5. **环境评估** — 环保合规风险

### 商业尽职调查 Commercial DD
1. **矿权审查** — 采矿权有效性和续期
2. **法律合规** — 许可证状态、诉讼
3. **税务** — 税务结构和风险
4. **合同** — 关键合同条款
5. **保险** — 保险覆盖范围

### 运营尽职调查 Operational DD
1. **人员** — 关键人才和劳工关系
2. **HSE** — 安全记录和环保合规
3. **社区** — 社区关系和SLO
4. **供应链** — 关键供应商和物流
5. **基建** — 道路、电力、水源

## 矿业M&A策略 M&A Strategy

### 收购动机 Rationale
| 动机 | 例子 |
|---|---|
| 规模扩张 | Newmont + Newcrest → 全球第一 |
| 地区协同 | Barrick + Randgold → 非洲协同 |
| 管线补充 | 老牌公司收购勘探成果 |
| 成本降低 | 合并后削减重叠的G&A |
| 估值套利 | 低迷市场中收购被低估资产 |

### PMH的M&A思考
- 作为junior miner，更可能是**被收购方**或**参与JV**
- 关键: 创造被收购价值 — 有高质量资源量 + 清晰的开发路径
- JV策略: 找大型公司作为合作伙伴，利用他们的资金和运营能力
- 案例: B2Gold收购Fekola项目 → 从勘探到生产，创造了巨大价值

---

*估值核心: DCF最严谨, EV/oz最直观, 可比交易最实用。PMH应关注如何在勘探阶段创造最大的每盎司价值。*
