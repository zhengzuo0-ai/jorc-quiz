# JORC Table 1 详解 — 报告清单 Table 1 Explained

*官方文件: [JORC Code 2012 — Table 1](https://www.jorc.org/docs/JORC_code_2012.pdf)*

## Table 1 是什么? What is Table 1?

> **通俗理解**: Table 1 就像一份"体检报告模板"。当你要向投资者报告你的矿的信息时，JORC Code说："你必须按照这份清单逐项汇报，漏了哪项必须解释为什么省略。" 这就是所谓的 **"如果没有，为什么没有" (If Not, Why Not)** 原则。

Table 1 是JORC Code 2012附录中的**评估与报告标准清单**，分为五个部分 (Section):

| Section | 标题 | 适用于 |
|---|---|---|
| **Section 1** | 取样技术与数据 | 所有报告 |
| **Section 2** | 勘探结果报告 | 勘探结果 |
| **Section 3** | 资源量估算与报告 | 资源量 |
| **Section 4** | 储量估算与报告 | 储量 |
| **Section 5** | 钻石和工业矿物 | 特殊矿种 |

## Section 1: 取样技术与数据 Sampling Techniques & Data

**每次报告都必须涵盖的基础信息:**

### 关键项目

| 项目 Item | 要求报告的内容 | 为什么重要 |
|---|---|---|
| **取样方法** Sampling | DD/RC/trenching等 | 投资者需要知道数据怎么来的 |
| **取样技术** Drilling Techniques | 钻孔类型、直径、深度 | 影响样品代表性 |
| **回收率** Drill Sample Recovery | 岩心/岩屑回收率 | 低回收率=可能丢失高品位 |
| **编录** Logging | 地质编录方法和详细程度 | 影响地质解释的可靠性 |
| **取样间距** Sub-sampling | 如何切割岩心、分样 | 直接影响分析结果 |
| **QA/QC** Quality of Assay | 标准样、空白样、重复样 | 数据可不可靠的关键证据 |
| **样品安全** Sample Security | 运输链管理 | 防止样品被篡改 |
| **化验方法** Assay | Fire Assay/AAS/ICP等 | 不同方法精度不同 |

## Section 2: 勘探结果报告 Exploration Results

### 最易出错的部分 (考试重点)

| 项目 | 要求 | 常见错误 |
|---|---|---|
| **数据汇总方法** | 说明品位加权方法、截断值 | 不说明截断品位 |
| **真厚度** vs 视厚度 | 明确区分 | 报告钻孔截长混淆为矿体厚度 |
| **平衡报告** | 报告所有结果，包括差的 | 只报告好的钻孔 |
| **图件** | 提供足够的地图和截面 | 缺少位置图、截面图 |

> **通俗理解**: 你不能只告诉投资者"我们钻到了20米@15g/t的金矿化！"而不说明这是真厚度还是视厚度、旁边的钻孔是不是空的、你用了什么截断品位。

## Section 3: 资源量估算 Resource Estimation

### 关键报告项目

| 项目 | 要求 |
|---|---|
| **数据库完整性** | 数据验证方法 |
| **地质解释** | 矿化模型的可信度 |
| **估算方法** | 克里金/IDW/其他 |
| **切割品位** | 高值是否被截断，如何截断 |
| **验证** | 模型是否与原始数据吻合 |
| **分类** | Measured/Indicated/Inferred的依据 |
| **密度** | 如何确定矿石密度 |
| **审计** | 是否有独立审查 |

## Section 4: 储量估算 Reserve Estimation

### 修正因素 Modifying Factors

> **通俗理解**: 资源量是"地下有多少金子"，储量是"你实际能拿出来卖钱的有多少"。中间的差距就是修正因素——开采损失、选矿回收不完全、经济上不划算的部分等。

| 修正因素 | 考虑内容 | 典型损失 |
|---|---|---|
| **采矿** Mining | 矿石损失率、贫化率 | 5-20% 损失 |
| **冶金** Metallurgy | 选矿回收率 | 5-15% 损失 |
| **经济** Economic | 金属价格、成本 | 低品位部分被排除 |
| **环境** Environmental | 限制区域 | 某些区域不能开采 |
| **社会** Social | 社区反对 | 某些区域受限 |
| **基建** Infrastructure | 道路、电力、水 | 偏远地区成本高 |

## "If Not, Why Not" 原则

JORC Code 2012 的核心报告原则:

- Table 1 中的每一项都需要被**考虑**
- 如果某项不适用，必须**解释为什么**
- 不能简单地跳过
- 目的: 确保投资者不会因为信息缺失而被误导

### 例子

```
❌ 错误: Table 1中不提及QA/QC
✅ 正确: "QA/QC: 本次勘探未系统插入标准样，因为是早期踏勘阶段。
         后续详查阶段将实施完整的QA/QC方案。"
```

---

*考试重点: Table 1不是可选的，是**强制性的**。每一项都要回答或解释为何不回答。勘探结果报告中最常见的错误是: 不说明截断品位、混淆真厚度和视厚度、选择性报告。*
