# Table 1 采样与数据 | Sampling Techniques & Data

> 💡 **"垃圾进，垃圾出"——资源量估算的质量完全取决于采样和数据的质量。JORC Table 1是确保数据透明和可验证的核心工具。**

## 核心概念 (Core Concepts)

### 采样类型详解

不同的采样方法适用于不同的勘探阶段和目的，每种方法都有其优势和局限性：

#### RC钻 (Reverse Circulation Drilling)

- **原理**：压缩空气将岩屑通过内管反向吹至地表收集，避免与孔壁接触造成的污染
- **成本**：**$80-150/米**（因地区、深度和地质条件而异）
- **优势**：速度快（每日可钻50-100米），成本低，适合大规模勘探
- **局限**：
  - 样品为岩屑而非完整岩芯，无法做详细地质记录
  - 含水层区域样品质量严重下降——地下水导致岩屑混合和品位偏差
  - 不适合做冶金试验（样品量和完整性不足）
  - 深孔（>300米）效率和质量下降
- **适用场景**：资源量定义钻探、区域勘探、异常验证
- **关键质量指标**：样品回收率、湿样比例、样品重量一致性

#### 金刚石岩芯钻 (Diamond Core Drilling, DD)

- **原理**：使用镶嵌金刚石的钻头切割完整的圆柱形岩芯
- **成本**：**$150-400/米**（因岩芯直径、深度和岩石硬度而异）
- **常见岩芯直径**：NQ (47.6mm), HQ (63.5mm), PQ (85.0mm)
- **优势**：
  - 提供完整岩芯，可做详细地质记录（结构、矿物、蚀变）
  - 岩芯可用于冶金/选矿试验
  - 可做定向测量和地质力学测试
  - 深孔质量稳定
- **局限**：成本高、速度慢（每日15-30米）、需要水源
- **适用场景**：资源量升级钻探、冶金样采集、地质构造研究、深部矿体勘探
- **关键质量指标**：岩芯回收率 (Core Recovery %)、RQD (Rock Quality Designation)

#### 渠道取样 (Channel Sampling)

- **原理**：在已暴露的岩石表面（如地下矿巷道壁或露头）切割连续的沟槽并收集样品
- **沟槽宽度**：通常 **10-20厘米**，深度2-5厘米
- **优势**：在已有巷道中成本极低，直接采集矿体横断面样品
- **局限**：仅限于已有暴露面，受取样者偏好影响
- **适用场景**：地下矿已开挖区域的品位验证，露头采样
- **关键要求**：必须连续取样，不能选择性跳过低品位区段

#### 抓取样 (Grab Sampling)

- **原理**：直接从岩石露头或矿堆中随机抓取样品
- **关键限制**：**不能用于资源量估算**
- **品位偏差**：取样者倾向于选择看起来矿化程度高的岩石，导致系统性高估
- **适用场景**：仅用于初步地质评估、发现矿化类型、快速判断矿化潜力
- **在报告中的披露**：必须明确标注为抓取样，并说明其不具备品位代表性

#### 土壤样 (Soil Sampling)

- **原理**：在地表以网格模式系统采集土壤样品，分析其中的微量元素含量
- **采样深度**：通常20-40厘米（B层位）
- **适用场景**：地球化学勘查的基础工具，用于发现矿化异常区域 (anomalies)
- **网格间距**：根据目标类型，从50×50米到400×100米不等
- **分析方法**：多元素ICP分析 (Inductively Coupled Plasma)
- **优势**：覆盖面积大，成本低，可快速圈定靶区
- **局限**：仅反映地表风化层的地球化学信息，不直接指示矿体品位和厚度

### QA/QC程序详解 (Quality Assurance / Quality Control)

QA/QC是保证采样和分析数据质量的系统性程序，是JORC Table 1的核心要求。没有完善的QA/QC，资源量估算缺乏可信度。

#### 空白样 (Blanks)

- **目的**：检测实验室是否存在**交叉污染** (cross-contamination)
- **材料**：使用已知不含目标元素的无矿化岩石样品
- **插入频率**：通常每 **20-30个样品** 插入1个
- **判断标准**：空白样的分析结果应低于检测限 (detection limit) 或接近零值
- **异常处理**：如空白样检出显著品位，说明实验室存在污染问题，必须调查并可能要求重新分析相关批次

#### 标准样 (CRMs - Certified Reference Materials)

- **目的**：检测实验室分析结果的**准确度** (accuracy)
- **材料**：经认证的、已知品位的标准参考物质，通常从专业供应商购买
- **插入频率**：通常每 **20-30个样品** 插入1个
- **判断标准**：分析结果应在认证值的±2个标准差范围内
- **最佳实践**：使用涵盖不同品位范围的多个标准样（低品位、中品位、高品位），以检验全品位范围的准确度
- **异常处理**：如标准样系统偏高或偏低，可能存在分析方法偏差，需要与实验室沟通校正

#### 副样 / 现场重复样 (Field Duplicates)

- **目的**：检测**采样代表性**和采样过程的精密度 (precision)
- **方法**：在同一采样点收集第二份平行样品
- **插入频率**：通常每 **20个样品** 1个（约5%）
- **判断标准**：副样与原样的相对差异应在可接受范围内（通常HARD值<20-30%）
- **意义**：高变异性可能反映矿化的"块金效应" (nugget effect) 或采样方法问题

#### 实验室内副样 (Lab Duplicates / Repeats)

- **目的**：检测实验室内部的分析**精密度** (precision / reproducibility)
- **方法**：实验室对同一粉碎样品进行两次独立分析
- **频率**：实验室通常自行执行，约每10-20个样品1个
- **判断标准**：两次分析结果的差异应在分析方法的固有精密度范围内

#### 第三方复检 (Umpire / Check Assays)

- **目的**：独立验证主实验室的分析结果，检测是否存在**系统性偏差**
- **方法**：将 **5-10%** 的粗碎副样或纸浆副样送至独立的第三方实验室分析
- **选择第三方实验室**：应为ISO 17025认证的独立实验室，与主实验室无商业关联
- **判断标准**：两个实验室的结果应在统计学上一致（无显著系统偏差）
- **重要性**：是JORC合规的关键要求，也是投资者和独立审核人最关注的QA/QC环节

### 样品制备流程 (Sample Preparation)

从矿场到最终分析结果，样品经历一个标准化的制备过程：

```
原始样品 (Field Sample)
    ↓
粉碎 (Crushing): 将大块样品粉碎至约2-6mm
    ↓
缩分 (Splitting): 使用旋转缩分器或分样器取出代表性子样
    ↓                 (通常取1-3kg)
研磨 (Pulverizing): 将子样研磨至75-150微米 (200目)
    ↓                  (>85%通过75微米筛)
分析 (Assaying): 取出分析样进行化学分析
```

每个步骤都有明确的质量标准：
- **粉碎**：必须确保粒度均匀，避免金颗粒损失
- **缩分**：旋转缩分器 (Rotary Splitter) 优于手工缩分，代表性更好
- **研磨**：研磨细度直接影响分析结果的精密度，尤其是含粗粒金的样品
- **样品量递减**：从数十公斤的原始样品逐步缩减到30-50克的分析样——确保每一步的代表性至关重要

### 火试金法 (Fire Assay)

火试金法 (Fire Assay) 是黄金分析的**行业标准方法**，几乎所有JORC合规的金矿资源量估算都基于此方法。

- **原理**：将样品与助熔剂 (flux) 混合后高温熔融，铅收集金（和银），经灰吹 (cupellation) 分离后称重或用AAS/ICP测定
- **标准取样量**：**30-50克**（通常为50克，称为"50克火试金"）
- **为什么不用更大取样量**：成本考虑；但对粗粒金矿 (coarse gold)，可使用更大取样量 (如1kg的锤式回收法 Leachwell/Screen Fire Assay)
- **精度**：对于正常品位范围的金矿样品，精度和准确度均优于其他方法
- **替代方法**：氰化瓶滚法 (Bottle Roll Cyanide Leach) 用于大样量检验，但标准分析仍以火试金为主
- **粗粒金效应 (Nugget Effect)**：如果金以粗粒形式存在，30-50克的取样量可能不具代表性，导致品位估算的高变异性——这是金矿勘探中最大的采样挑战之一

### 数据管理 (Data Management)

现代矿业勘探依赖专业的数据库系统确保数据完整性和可追溯性：

#### 专业数据库系统

- **acQuire**：澳大利亚开发的行业标准地质数据库系统，广泛用于JORC合规项目
- **Datamine / Fusion**：集成地质建模和数据管理功能
- **Micromine / Geobank**：另一款常用的地质数据管理平台
- **Geovia GEMS**：达索系统旗下的综合矿业软件套件

#### 数据验证 (Data Validation)

- 钻孔深度 (downhole depth) 验证：确保深度间隔无重叠或缺口
- 岩芯/岩屑记录与分析结果的交叉核对
- 孔口坐标测量和偏斜测量 (downhole survey) 的完整性
- 数据输入的双盲校验 (double-entry verification)

#### QA/QC监控图表

- **控制图 (Control Charts)**：绘制标准样和空白样的分析结果随时间的变化趋势
- **Thompson-Howarth图**：评估副样精密度
- **散点图 (Scatter Plots)**：对比主实验室与第三方实验室的结果
- **偏差分析**：计算均值偏差 (mean bias) 和相对标准差 (RSD)
- 这些图表必须定期审查，异常必须及时追踪和纠正

## PMH 实践关联 (PMH Context)

> 在PMH的西非和纳米比亚勘探项目中，采样和数据质量管理面临独特挑战：
>
> - **RC钻在西非的普遍应用**：由于成本效率高，RC钻通常是首选勘探方法，但西非雨季期间的高水位可能严重影响深部样品质量——必须仔细记录湿样并评估对品位的影响
> - **金刚石岩芯钻的战略使用**：在品位升级和冶金试验阶段转用DD钻，特别是对于需要确认氧化/硫化矿过渡深度的项目
> - **实验室选择**：西非项目通常使用经ISO认证的区域实验室（如SGS、ALS在西非的分支机构），第三方复检则可送至南非或澳大利亚的独立实验室
> - **土壤采样的有效性**：西非大部分地区有深厚的红土风化层 (laterite)，传统土壤地球化学方法可能被风化层掩盖，需要考虑更深的采样层位或使用气体地球化学等替代方法
> - **数据管理的挑战**：偏远矿区的数据传输和备份需要特别关注，防止数据丢失
