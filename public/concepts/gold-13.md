# 矿业法规与合规 Mining Regulations & Compliance

## 澳大利亚矿业法规框架 Australian Mining Regulatory Framework

### 联邦层面 Federal Level
| 法规 Legislation | 管辖 | 关键要求 |
|---|---|---|
| EPBC Act 1999 | 环境保护 | 国家重要环境事项审批 |
| Native Title Act 1993 | 原住民权益 | 原住民土地权利保护 |
| Corporations Act 2001 | 公司治理 | 上市公司披露要求 |
| ASX Listing Rules | 证券交易 | 矿业报告必须遵循JORC |

### 州/领地层面 State/Territory Level
各州有各自的矿业法:
- **Mining Act** — 矿权管理
- **Environmental Protection Act** — 环境审批
- **Aboriginal Heritage Act** — 文化遗产保护

### ASX 上市规则与JORC Code

**ASX Listing Rule Chapter 5** 要求:
- 所有矿业公开报告必须遵循 **JORC Code 2012**
- 资源量/储量变更必须及时公告
- 勘探结果报告必须由胜任人签署
- 不得发布误导性信息

## 矿权制度 Tenure System

### 矿权类型 Types of Mining Tenure

| 类型 Type | 允许活动 | 期限 | 特点 |
|---|---|---|---|
| 勘探许可证 Exploration License (EL) | 勘探 | 2-5年，可续 | 面积较大 |
| 保留许可证 Retention License (RL) | 保留权利 | 5年 | 暂不开发但保留权利 |
| 采矿许可证 Mining Lease (ML) | 开采 | 21-25年，可续 | 需环评和开发计划 |
| 矿产开发许可证 Mineral Development License | 评估 | 5年 | 可行性研究阶段 |

### 矿权义务 Tenure Obligations
- **最低勘探支出 Minimum Expenditure** — 每年必须在矿权区投入一定金额
- **报告义务 Reporting** — 定期提交勘探报告
- **环境保证金 Environmental Bond** — 缴纳保证金用于复垦
- **原住民协商 Native Title** — 与原住民权利持有人协商

## 环境审批流程 Environmental Approval Process

### 典型流程
```
环境影响评估 (EIA) → 公众咨询 → 政府审批 → 环境管理计划 → 监测与合规
```

### 关键文件
1. **环境影响评估报告 EIS/EIA** — 项目环境影响全面评估
2. **环境管理计划 EMP** — 运营期间环境管理措施
3. **矿山闭坑计划 Mine Closure Plan** — 闭矿后的复垦和监测
4. **水资源管理计划 Water Management Plan** — 用水和排水管理

## 国际矿业报告标准 International Reporting Standards

| 标准 Standard | 地区 Region | 关键特点 |
|---|---|---|
| JORC Code 2012 | 澳大利亚/亚太 | 胜任人制度 |
| NI 43-101 | 加拿大 | 合格人 (QP) 制度 |
| SAMREC Code | 南非 | 类似JORC |
| SEC S-K 1300 | 美国 | 2018年起与国际接轨 |
| PERC Standard | 欧洲 | 类似JORC |
| CRIRSCO | 国际 | 各标准的协调框架 |

### CRIRSCO 国际协调
- Committee for Mineral Reserves International Reporting Standards
- 确保各国标准互相对应
- 主要差异在程序和法律框架，技术原则趋于一致

## 社区协商与原住民权利 Community & Indigenous Rights

### Free, Prior and Informed Consent (FPIC)
- 自由: 不受胁迫
- 事先: 在项目开始前
- 知情: 提供完整信息
- 同意: 获得社区许可

### 原住民文化遗产保护
- 所有地面活动前需要**文化遗产调查 (Heritage Survey)**
- 不得破坏注册的原住民遗址
- 需要**原住民代表参与**调查过程

---

*合规要点: JORC是技术标准，ASX是法律要求，环评是必经程序，原住民协商是社会义务。四者缺一不可。*
