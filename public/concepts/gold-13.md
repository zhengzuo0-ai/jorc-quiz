# 钻探 | Drilling

> 💡 **钻探是将地表的地质推测转化为地下事实的唯一方法——它是勘探预算的最大单项支出，也是资源量估算的数据基础。理解钻探方法、设计和质量控制，是CEO评估勘探报告可信度的关键。**

## 核心概念 (Core Concepts)

### 1. 两种主要钻探方法 (RC vs Diamond Core)

金矿勘探中最常用的两种钻探方法各有优劣，理解它们的区别是评估勘探计划的基础。

#### 反循环钻探 (RC Drilling, Reverse Circulation)

**原理：** 使用压缩空气驱动锤头在井底破碎岩石，碎岩屑随气流通过双壁钻杆的内管上升到地表收集。称为"反循环"是因为空气从钻杆外管向下输送，岩屑从内管向上返回——与常规钻探中冲洗液带着岩屑从钻杆外围上返相反。

**关键参数：**

| 参数 | 典型范围 | 说明 |
|------|----------|------|
| 孔径 | 5.5 英寸 (140mm) | 标准RC钻头直径 |
| 最大深度 | 300-500m | 受空压机能力限制；深度越大效率越低 |
| 进尺速度 | 50-150m/天 | 取决于岩石硬度和地下水条件 |
| 成本 | $80-150/m | 西非典型范围（含动复迁费） |
| 样品形式 | 岩屑 (Chips) | 每米或每2米采集一个样品（约3-5 kg） |

**优势：**
- 速度快、成本低——同样预算可打更多米数
- 样品代表性好（理论上每米等量采样）
- 适合初步验证和资源量界定的大间距钻探
- 在西非是金矿勘探最常用的方法

**劣势：**
- 获取的是碎岩屑而非完整岩芯，地质结构信息有限
- 在地下水位以下，样品可能被污染（水将岩屑冲混）——称为"湿样问题"(Wet Sample Issue)
- 不适合冶金试验和地质力学研究
- 倾斜角度有限（通常 -60° 到 -90°）

#### 金刚石钻探 (Diamond Core Drilling)

**原理：** 使用镶嵌工业金刚石的环形钻头，在岩石中切出圆柱形岩芯 (Core)。岩芯通过岩芯管 (Core Barrel) 提升到地表，提供完整的连续岩石样本。

**关键参数：**

| 参数 | 典型范围 | 说明 |
|------|----------|------|
| 岩芯直径 | NQ (47.6mm) / HQ (63.5mm) | NQ更常用，HQ用于需要更大样本时 |
| 最大深度 | 1,000m+ | 技术上可超过3,000m |
| 进尺速度 | 20-60m/天 | 比RC慢得多 |
| 成本 | $150-350/m | 约为RC的2-3倍 |
| 样品形式 | 完整岩芯 (Core) | 连续的圆柱形岩石样本 |

**优势：**
- 提供完整的地质信息：岩性、矿化、蚀变、构造（断层、剪切带、褶皱）
- 岩芯可用于冶金试验 (Metallurgical Testwork) 和地质力学测试 (Geotechnical Testing)
- 深度不受限
- 可定向钻探 (Directional Drilling)，灵活调整角度

**劣势：**
- 成本高、速度慢
- 岩芯回收率可能不理想（破碎带、断层带）
- 需要水作为冲洗液——在干旱区域可能需要额外供水

#### 何时使用哪种方法？

| 场景 | 推荐方法 | 原因 |
|------|----------|------|
| 初步靶区验证 | RC | 快速、低成本获取品位信息 |
| 资源量界定（宽间距） | RC | 经济高效地覆盖大面积 |
| 资源量升级（加密） | RC + DD 组合 | RC定界，DD提供地质细节 |
| 深部勘探 (>300m) | DD | RC无法有效到达 |
| 可行性研究支撑 | DD | 需要冶金和地质力学数据 |
| 氧化带/风化带勘探 | RC | 氧化带通常较浅 |

### 2. 钻孔设计 (Drill Hole Design)

钻孔设计是勘探方案的核心，直接决定了钻探数据的质量和资源量估算的可靠性。

#### 关键设计参数

**方位角 (Azimuth)**
- 钻孔在水平面上的方向，以北为0°，顺时针计量
- **核心原则**：钻孔应尽量**垂直穿过**矿体的走向 (Strike)，以获得矿体的真厚度 (True Thickness)
- 如矿体走向为 N-S (0°/180°)，钻孔方位应设计为 E-W (90° 或 270°)
- 如果钻孔平行于矿体走向，截获的厚度将被夸大，给出误导性的结果

**倾角 (Dip)**
- 钻孔与水平面的夹角
- 通常设计为 -50° 到 -70°（向下倾斜）
- 必须考虑矿体的倾角——钻孔应以较大角度截穿矿体
- 垂直孔 (-90°) 在矿体倾角陡时截获厚度可能被低估

**深度 (Depth)**
- 根据靶区深度和勘探目标确定
- RC 通常 100-300m，DD 可达 500-1,000m+
- 钻孔应穿过矿体后继续延伸至少 20-30m，确保完全截穿

**间距 (Spacing)**
- 相邻钻孔之间的距离，直接影响资源量的分类级别

| 资源量级别 | 典型钻探间距 | JORC 要求 |
|-----------|-------------|-----------|
| 推断 (Inferred) | 100-200m | 地质和品位连续性有合理推断 |
| 指示 (Indicated) | 50-100m | 地质和品位连续性有合理假定 |
| 探明 (Measured) | 25-50m | 地质和品位连续性有高度确信 |

> **注意：** 上述间距只是一般指导。实际间距取决于矿化类型的连续性——连续性好的矿体（如层控型）可以用更大间距，连续性差的矿体（如含粗粒金的石英脉）需要更密的间距。

### 3. 岩芯编录 (Core Logging)

岩芯编录是将钻取的岩芯转化为地质信息的过程，由地质师在岩芯棚 (Core Yard) 中进行。

**编录内容：**

| 记录项 | 英文 | 说明 |
|--------|------|------|
| 岩性 | Lithology | 岩石类型：花岗岩、片岩、砂岩等 |
| 矿化 | Mineralization | 可见金、硫化物（黄铁矿/毒砂）的类型、比例、分布 |
| 蚀变 | Alteration | 硅化、碳酸盐化、绢云母化等蚀变类型和强度 |
| 构造 | Structure | 断层、剪切带、褶皱、节理的位置和角度 |
| 石英脉 | Quartz Veins | 脉体的宽度、方向、与矿化的关系 |
| 岩芯回收率 | Core Recovery (%) | 实际回收的岩芯长度/钻进长度——低回收率意味着数据缺失 |
| RQD | Rock Quality Designation | 长度≥10cm的完整岩芯段长度总和/钻进长度——衡量岩体质量 |
| 地质力学 | Geotechnical | 用于矿山设计的岩石强度和节理数据 |

**岩芯照片：** 所有岩芯在编录后拍照存档，作为永久性记录。干湿两种条件下拍照——湿照片更容易识别矿化和蚀变特征。

### 4. 采样与化验 (Sampling & Assaying)

#### 岩芯切割与采样

1. **标记采样区间**：通常每 1m 一个样品，矿化带可加密至 0.5m
2. **岩芯切割**：使用金刚石锯片将岩芯沿轴线切成两半
3. **半芯采样**：一半送化验室分析，另一半保留在岩芯箱中作为参考
4. **样品称重和编号**：每个样品独立编号，与钻孔数据库对应

#### RC 样品采集

1. 每米或每2米通过旋转分样器 (Rotary Splitter) 从全部岩屑中分出代表性样品
2. 样品重量约 3-5 kg
3. 每米一份现场地质记录（岩屑描述、颜色、湿度）

#### 化验方法

| 方法 | 英文 | 适用 | 检测限 |
|------|------|------|--------|
| 火试金 | Fire Assay | 金的标准化验方法 | 0.01 ppm |
| 湿法化学 | Aqua Regia Digest | 多元素普查 | 1-10 ppb |
| ICP-MS | 等离子体质谱 | 路径元素和痕量元素 | 0.1-1 ppb |
| 瓶滚浸出 | Bottle Roll Leach | 冶金回收率初步测试 | — |

**火试金 (Fire Assay)** 是金化验的"黄金标准"——将 30g 或 50g 样品与铅和助熔剂混合在高温坩埚中熔融，金被铅收集，再通过酸溶解铅后用原子吸收光谱法 (AAS) 或 ICP 测定金含量。

### 5. 钻探 QA/QC (Quality Assurance / Quality Control)

QA/QC 是确保钻探数据可靠性的系统化程序，是 JORC 合规报告的必要条件。没有完善的 QA/QC，资源量估算的可信度将大打折扣。

#### 三种核心控制样品

**标准样品 (Certified Reference Materials, CRMs / Standards)**
- 已知品位的样品，由第三方机构认证
- 插入频率：每 20-25 个样品插入 1 个（约 4-5%）
- **用途**：检测化验室的准确度 (Accuracy)——化验结果应在标准值的 ±10% 以内
- 应使用多个不同品位的标准样品（低品位、中品位、高品位）
- 如果标准样品结果持续偏离预期值，说明化验室存在系统性误差，需要调查原因

**空白样品 (Blanks)**
- 已知不含金（或含金量极低）的样品
- 插入频率：每 20-25 个样品插入 1 个
- **用途**：检测样品间的交叉污染 (Cross-contamination)——空白样品的金含量应 < 0.1 ppm
- 特别应在高品位样品**之后**插入空白——如果空白样品出现异常高值，说明前一个高品位样品污染了后续样品
- 这在含粗粒金的矿床中尤其重要

**重复样品 (Duplicates / Twins)**
- 从同一采样点取两份样品，分别送化验
- 包括：
  - **现场重复 (Field Duplicates)**：从同一采样区间取第二份样品
  - **实验室重复 (Lab Duplicates)**：化验室内部对同一样品重复分析
  - **加密检查 (Umpire Assays)**：将部分样品送至第二家独立化验室分析
- 插入频率：5-10%
- **用途**：检测精密度 (Precision)——两个结果之间的差异应在可接受范围内

#### QA/QC 数据监控

```
合格的 QA/QC 表现：
- 标准样品：95%+ 的结果在预期值 ±2 个标准差以内
- 空白样品：95%+ 的结果在可接受限值以下
- 重复样品：配对分析的半绝对相对偏差 (HARD) < 20%
```

### 6. 井下测量 (Downhole Surveys)

钻孔在地下并不沿直线行进——受地层硬度变化、钻机震动等因素影响，钻孔会偏离设计轨迹。井下测量用于确定钻孔在不同深度的实际位置。

**测量方法：**

| 方法 | 英文 | 精度 | 适用 |
|------|------|------|------|
| 单点测量 | Single Shot | ±1° | RC钻孔，基础定位 |
| 多点测量 | Multishot (Reflex EZ-Shot) | ±0.5° | RC/DD，常规使用 |
| 陀螺仪 | Gyroscopic Survey | ±0.1° | DD，高精度需求，磁性岩层 |

**测量频率：** 通常每 30-50m 测量一次，在偏差敏感区域可加密至每 10-20m。

**为什么重要？** 如果一个 300m 深的钻孔偏离设计方位 5°，其井底位置可能偏移 26m——这足以导致矿化截获位置的严重误判，影响三维地质模型和资源量估算。

### 7. 钻探水管理 (Water Management)

在西非的雨季和含水层丰富地区，水管理是钻探作业的重大挑战：

- **RC 钻探**：地下水会降低空气冲击力，降低效率并导致样品质量下降（湿样品可能混合不同深度的矿化信息）
- **DD 钻探**：需要水作为冲洗液，但水量需要控制，过多导致岩芯冲蚀
- **排水设施**：钻探平台需要修建沉淀池和排水沟
- **旱季优势**：西非多数勘探公司将主要钻探活动安排在旱季（11月-4月），以获得最佳样品质量

### 8. 钻探承包商管理 (Drill Contractor Management)

Junior 矿企通常不拥有钻机——而是聘请专业钻探承包商：

**西非主要钻探承包商：**
- **Capital Drilling**：在非洲和中亚运营，ASX上市
- **Geodrill**：总部在加纳，专注西非，TSX上市
- **Master Drilling**：南非公司，专注大口径和定向钻探
- **Foraco**：法国公司，在西非有强大的存在

**合同关键条款：**

| 条款 | 说明 |
|------|------|
| 计价方式 | 按进尺 (Per Meter) 或按时间 (Day Rate) |
| 动复迁费 | Mobilization/Demobilization Fee：将钻机运至/运离现场的费用（$30,000-100,000/次） |
| 最低保证米数 | Minimum Guaranteed Meterage：保证承包商的最低收入 |
| 待机费 | Standby Rate：因天气、设备故障等原因停工时的日费率 |
| 安全指标 | LTIFR (Lost Time Injury Frequency Rate) 要求 |
| 样品质量条款 | 对岩屑/岩芯回收率的最低要求 |

**管理要点：**
- 建立日报制度——每日汇总进尺、样品质量、安全和设备状况
- 现场监督：至少配备一名经验丰富的高级地质师 (Senior Geologist) 常驻
- 定期检查样品质量——RC 的分样器清洁度、DD 的岩芯回收率
- 管理好承包商排期——西非旱季是钻探旺季，承包商的钻机资源紧张

## PMH 实践关联 (PMH Context)

> 钻探是 PMH 勘探预算中最大的单项支出，通常占勘探费用的 60-80%。以下是关键实践考量：
>
> **钻探策略**：PMH 在西非的早期靶区验证应以 RC 为主（成本低、速度快），取得鼓励性结果后再用 DD 进行加密和技术研究。典型的 RC:DD 比例在早期为 80:20，进入可行性阶段后逐步提高 DD 比例至 50:50。
>
> **季节性安排**：西非的旱季（约 11 月至次年 4 月）是最佳钻探季节——地面可达性好、RC 样品质量高（少湿样）、施工效率高。PMH 应在雨季完成数据分析和钻探方案设计，旱季一开始就启动钻探。
>
> **QA/QC 合规**：如果 PMH 计划按 JORC 标准发布资源量，完善的 QA/QC 程序从第一个钻孔就必须到位。补救性的 QA/QC 成本远高于从一开始就做对。
>
> **承包商关系**：在西非旱季钻探旺季，钻机资源非常紧张。PMH 应提前 3-6 个月锁定承包商合同，避免旺季无钻机可用。与 Geodrill 或 Capital Drilling 建立长期合作关系有利于保障资源和控制成本。
>
> **成本管理**：对于年度钻探预算 $2-5M 的 Junior 公司，每米成本差 $10 就意味着数千米进尺的差异。精细的钻孔设计——避免过深的无效孔段、优化孔位布置——是控制成本的关键。
