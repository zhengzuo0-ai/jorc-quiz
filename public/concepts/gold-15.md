# 科特迪瓦黄金 | Côte d'Ivoire Gold Mining

> 💡 **科特迪瓦（象牙海岸）正迅速崛起为西非最具活力的黄金生产国。2024年Endeavour Mining的Lafigué矿山投产、Perseus Mining运营Sissingué和Yaouré矿山、Barrick的Tongon矿持续生产，使该国黄金年产量有望在2025年创下历史新高。坐拥世界级比里米安绿岩带地质条件、2014年矿业法（Code Minier 2014）投资友好框架和多家国际矿企的成熟运营，科特迪瓦正定位为西非黄金版图中的关键一极。理解该国的地质背景、矿业法规和关键项目，是评估西非黄金资产的核心能力。**

## 核心概念 (Core Concepts)

### 1. 地质背景 (Geological Setting)

科特迪瓦的黄金潜力几乎完全源于古元古代比里米安超群 (Paleoproterozoic Birimian Supergroup)，年龄约2.0–2.2 Ga。比里米安地层是西非克拉通 (West African Craton) 内最重要的含金建造，从塞内加尔经几内亚、马里、布基纳法索、加纳延伸至科特迪瓦，构成了世界上最大的前寒武纪绿岩带黄金省之一。

#### 比里米安绿岩带 (Birimian Greenstone Belts)

比里米安地层由两大岩石组合构成：

| 组合 | 岩性 | 含金意义 |
|------|------|----------|
| 火山岩带 (Volcanic Belts) | 玄武岩、安山岩、凝灰�ite；常伴有次火山侵入体 | 主要赋矿环境——剪切带控制的造山型金矿 |
| 沉积盆地 (Sedimentary Basins) | 灰岩、页岩、浊积岩 | 次要赋矿环境，盆地边缘构造带可含矿 |

科特迪瓦境内有多条北东—南西走向的绿岩带，被花岗岩-片麻岩穹窿 (Granite-Gneiss Domes) 分隔。黄金矿化主要受区域性剪切带 (Shear Zones) 控制，类型以造山型金矿 (Orogenic Gold) 为主，矿化产于含石英脉的韧性—脆韧性剪切带内，伴有绢云母-碳酸盐-黄铁矿蚀变组合 (Sericite-Carbonate-Pyrite Alteration Assemblage)。

#### 主要金矿带 (Key Gold Belts)

科特迪瓦已识别出多条高潜力金矿带：

**Ity-Floleu 带 (西南部)**
- 位于Toulépleu绿岩带，赋存世界级Ity金矿
- 矿化与铁镁质火山岩中的剪切带和品位富集带 (High-Grade Shoots) 相关
- 兼有氧化带 (Oxide) 和硫化物 (Sulphide) 矿石类型

**Tongon 带 (北部)**
- 位于Boundiali绿岩带 (Senoufo Belt)，赋存Tongon金矿
- 矿化产于北东走向的Nielle剪切带内
- 以石英脉型和剪切带蚀变岩型矿化为主

**Abujar 带 (中西部)**
- 位于Daloa绿岩带，赋存Abujar金矿
- 矿化与花岗闪长岩侵入体边缘的剪切带相关
- 较新发现——展示了科特迪瓦仍有重大发现潜力

**其他新兴区域：** Korhogo带、Boundiali-Tengrela走廊、Odienné地区——大量初级勘探公司正在活跃勘查。

### 2. 主要运营矿山 (Active Mines)

#### Ity 金矿 (Endeavour Mining)

Ity是科特迪瓦最大、最重要的金矿，也是西非排名前列的单体金矿之一。

| 参数 | 详情 |
|------|------|
| 运营商 | Endeavour Mining (原Endeavour收购La Mancha权益后全资控股) |
| 矿山类型 | 露天开采 (Open Pit) + CIL工厂 |
| 投产年份 | 原CIL工厂2019年投产（前身Heap Leach始于1991年） |
| 年产金量 | ~250,000-300,000 oz/year |
| 矿山寿命 | 储量支撑至2030年代中期，资源量进一步延伸 |
| 品位 | 储量品位约1.5-1.8 g/t Au |
| 处理能力 | ~5 Mtpa CIL工厂 |
| 特点 | 含显著氧化矿+硫化矿；Le Plaque、Daapleu、Walter等多个矿坑 |

Ity矿区面积大，持续勘探不断发现新矿体，使其成为一个长寿命的旗舰资产。

#### Lafigué 金矿 (Endeavour Mining) — 新投产

Lafigué是Endeavour Mining在科特迪瓦的第二座矿山，于**2024年10月正式投产**，标志着科特迪瓦黄金产业的重大扩张里程碑。

| 参数 | 详情 |
|------|------|
| 运营商 | Endeavour Mining |
| 矿山类型 | 露天开采 + CIL工厂 |
| 投产年份 | 2024年10月 |
| 目标年产金量 | **~200,000 oz/year**（2025年起达产） |
| 位置 | 科特迪瓦中北部 |
| 战略意义 | Endeavour在CDI的第二支柱资产；推动CDI黄金产量迈向历史新高 |

Lafigué的投产使Endeavour在科特迪瓦同时运营两座世界级矿山（Ity + Lafigué），年产金量合计可达450,000-500,000 oz。Endeavour 2024年西非总产量指导为**1.13-1.27百万盎司**（涵盖CDI、塞内加尔、布基纳法索等多国资产），CDI已成为其最重要的运营管辖区之一。受益于Lafigué的全年贡献，**CDI黄金总产量有望在2025年创下国家纪录**。

#### Tongon 金矿 (Barrick Gold)

| 参数 | 详情 |
|------|------|
| 运营商 | Barrick Gold (通过2019年Randgold Resources合并获得) |
| 矿山类型 | 露天开采 + CIL工厂 |
| 投产年份 | 2010年 |
| 年产金量 | ~200,000 oz/year（峰值）；2024年后产量逐步下降 |
| 矿山寿命 | 露天储量接近枯竭；正评估地下延伸可行性和周边勘探靶区 |
| 品位 | 储量品位约2.0-2.2 g/t Au |
| 特点 | 北南两个矿坑；位于Boundiali绿岩带Nielle剪切带 |

Tongon是原Randgold在科特迪瓦的旗舰项目，由Mark Bristow时代开发。矿山面临露天储量递减挑战，Barrick正评估地下开采可行性以延长矿山寿命。区域勘探潜力仍然可观——Tongon周边的绿岩带仍有待深入勘查。

Perseus Mining是科特迪瓦的重要中型矿商，在CDI运营两座金矿：**Sissingué** 和 **Yaouré**。

**Yaouré 金矿：**

Abujar的发现和快速开发是科特迪瓦近年最大的成功故事之一，证明该国仍有发现大型新矿床的潜力。Perseus通过收购Tietto获得了该资产，使其在CDI形成了**三矿山运营平台**（Sissingué + Yaouré + Abujar），进一步巩固了其作为非洲领先中型金矿商的地位。

### 3. 矿业法规框架 (Mining Code & Regulatory Framework)

#### 2014年矿业法 (Code Minier 2014)

科特迪瓦于2014年颁布了新矿业法 (Loi n° 2014-138)，替代了1995年旧法，旨在平衡吸引投资与提高国家收益。关键条款包括：

**税费结构 (Fiscal Terms)：**

| 税费项目 | 税率/条款 |
|----------|-----------|
| 企业所得税 (Corporate Tax) | 25%（矿业企业适用） |
| 黄金特许权使用费 (Royalty) | 3-5%（按金价阶梯浮动） |
| 国家免费持股 (State Free Carry) | 10%（国家在矿业公司中的免费权益） |
| 附加持股选择权 | 国家可按市场价额外购买最高15%股权 |
| 增值税 (VAT) | 标准18%，勘探阶段部分豁免 |
| 稳定条款 (Stability Clause) | 矿业协议签署后税制稳定期约10-15年 |

**特许权使用费阶梯 (Royalty Sliding Scale)：**
- 金价 ≤ $1,000/oz：3%
- 金价 $1,000-$1,300/oz：3.5%
- 金价 $1,300-$1,600/oz：4%
- 金价 > $1,600/oz：5%

注意：在当前金价（>$2,000/oz）环境下，实际适用税率为5%。政府已多次讨论修订阶梯上限，反映了资源民族主义 (Resource Nationalism) 的压力。

**许可证体系 (License System)：**
- 勘探许可证 (Permis de Recherche)：初始4年，可续期两次（每次3年），面积最大400 km²
- 开采许可证 (Permis d'Exploitation)：有效期最长20年，可续期
- 半工业许可证 (Permis d'Exploitation Semi-Industrielle)：针对小型矿山

#### 政府机构

| 机构 | 职能 |
|------|------|
| 矿业部 (Ministère des Mines) | 政策制定、许可证审批 |
| SODEMI | 国有矿业公司，管理国家权益和参与勘探 |
| DGMG | 地质和矿业总局，技术审批 |
| BNEDT | 国家技术研究和发展局 |

### 4. 关键企业与合资结构 (Key Players & JV Structures)

科特迪瓦的黄金行业由国际中大型矿企主导：

| 企业 | 主要资产 | 角色 |
|------|----------|------|
| Endeavour Mining | **Ity + Lafigué**（2024年10月投产） | CDI最大运营商；2024年西非总产量1.13-1.27M oz |
| Barrick Gold | Tongon | 超级大型矿企；评估地下延伸 |
| Perseus Mining | **Sissingué, Yaouré, Abujar** (原Tietto) | 中型矿商；CDI三矿山平台；2024年收购坦桑尼亚Nyanzaga |
| Allied Gold | Agbaou (原Endeavour出售) | 中型运营商 |
| Montage Gold | Koné项目 (开发阶段) | 开发阶段公司；>3 Moz资源量 |
| Predictive Discovery | Banankoro (几内亚/科特迪瓦边境) | 初级勘探阶段 |

**合资结构 (JV Structures) 特点：**
- 科特迪瓦政府通过SODEMI持有10%免费权益 (Free Carry)，不承担勘探和建设成本
- 国际矿企通常通过在科特迪瓦注册的子公司 (Filiale) 持有许可证
- 项目层面合资 (Project-Level JVs) 常见于勘探阶段，高级矿企通过"赚取权益"(Earn-In) 方式进入
- 矿企被要求在阿比让证券交易所 (BRVM) 上市部分股权——虽执行不严格但政策方向明确

### 5. PMH CDI 业务背景 (PMH CDI Operations Context)

PMH（Pacific Mining Hub 或相关实体）在科特迪瓦的黄金业务代表了初级到中级勘探公司的典型路径：

**Track A 勘探策略 (Track A Exploration)：**
- Track A 代表PMH在科特迪瓦的第一优先级勘探管道
- 典型策略：获取大面积勘探许可证 → 区域地球化学和地球物理普查 → 靶区筛选 → RAB/AC钻探验证 → RC钻探资源界定
- 重点关注未充分勘探的绿岩带区域，利用先行者优势 (First-Mover Advantage)
- 勘探预算通常$2-5M/年，主要用于钻探和地球化学分析

**运营挑战：**
- 雨季 (6-9月) 限制钻探窗口
- 社区关系 (Community Relations) 和地表权利 (Surface Rights) 谈判
- 手工采金 (Artisanal Mining / Orpaillage) 的管理——既是挑战也是找矿线索
- 合规要求：环境影响评估 (EIES)、社会责任 (RSE)、定期进度报告

**战略考量：**
- 科特迪瓦相对马里和布基纳法索具有政治稳定性优势 (Political Stability Premium)
- 阿比让是西非最发达的商业中心之一，后勤便利
- 法语区运营需要法语技术团队
- 成功的关键：将勘探发现推进到JORC资源量阶段，以支撑下一步融资或吸引合资伙伴

### 6. 市场与展望 (Market Outlook)

科特迪瓦的黄金产量从2010年的约5吨增长到2024年的约50吨以上，十多年间增长了10倍。**受益于Lafigué矿山2024年10月投产并于2025年全年达产，CDI黄金总产量有望在2025年创下历史新高。** 政府目标是成为非洲前五大黄金生产国，CDI正定位为**西非崛起中的黄金生产强国 (West Africa's Rising Gold Producer)**。

主要增长驱动力包括：

- **Lafigué矿山已投产**：Endeavour的新矿山于2024年10月投产，目标年产量200,000 oz（2025年起全年贡献）
- **Perseus三矿山平台**：Sissingué + Yaouré + Abujar在CDI的组合产量持续增长
- **Koné项目**：Montage Gold开发中的大型项目，资源量>3 Moz
- **棕地扩展 (Brownfield Expansion)**：Ity的持续扩张和周边卫星矿坑开发
- **绿地发现 (Greenfield Discovery)**：科特迪瓦仍有大面积地质条件优越但勘探不足的区域

**关键数据参考 (Key Data Points)：**
- Endeavour Mining 2024年西非总产量指导：**1.13-1.27百万盎司**（来源：Endeavour Mining 2024 Guidance）
- Perseus Mining CDI运营矿山：Sissingué (~80-100k oz/yr) + Yaouré (~200k+ oz/yr) + Abujar (~200k oz/yr设计产能)
- Barrick Tongon：约200k oz/yr（峰值），产量随储量递减
- 法律框架：**Code Minier 2014** (Loi n° 2014-138)，阶梯式特许权使用费3-5%

投资者应关注的风险包括：资源民族主义趋势、矿业法修订可能性（当前金价远超原阶梯上限）、手工采金冲突、以及区域安全形势（北部与布基纳法索边境的不稳定因素）。
