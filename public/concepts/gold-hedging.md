# 黄金对冲策略 Gold Hedging Strategies

*参考资料: [CME Group - Hedging Risk with Gold](https://www.cmegroup.com/articles/case-study/case-study-hedging-risk-with-gold-futures-and-options.html), [OCIM - Gold Hedging](https://www.ocim.com/insights/gold-hedging-a-critical-move-for-reducing-financial-uncertainty/), [Chatham Financial - Collars](https://www.chathamfinancial.com/insights/using-commodity-collars-to-manage-market-volatility), [BullionVault - Hedging Dilemma](https://www.bullionvault.com/gold-news/opinion-analysis/gold-silver-hedging-miners-dilemma-08262013)*

## 为什么矿企要对冲？ Why Do Miners Hedge?

> **通俗理解**: 想象你是一个果农，你的苹果要在6个月后收获。你不知道到时候苹果会卖什么价——可能涨到10元/斤，也可能跌到3元/斤。对冲就是现在就和买家签一个合同，锁定6元/斤的价格。这样你不会发大财（涨到10元你也只能卖6元），但也不会破产（跌到3元你还是卖6元）。矿企对冲金价是同样的道理。

**核心目的:**
- **减少现金流不确定性** — 让收入可预测
- **保护利润率** — 金价下跌时不至于亏损
- **提升信用评级** — 稳定的现金流让银行更愿意贷款
- **支持项目融资** — 银行通常要求新矿山项目做一定比例的对冲

## 三大对冲工具 Three Key Instruments

### 1. 远期合约 Forward Contracts

> **最简单直接**: 现在就约定6个月后以$X/oz的价格卖出一定数量的金。

```
例子:
- 当前金价: $4,000/oz
- 签远期合约: 6个月后以 $4,050/oz 卖出 10,000 oz
  (远期价格通常略高于现价，因为包含了存储和保险成本)

如果6个月后:
- 金价涨到 $5,000 → 你还是卖 $4,050 (少赚了$950/oz)
- 金价跌到 $3,000 → 你还是卖 $4,050 (多赚了$1,050/oz)
```

| 优势 | 劣势 |
|---|---|
| 无前期成本 | 完全放弃上涨收益 |
| 简单明确 | 价格锁死 |
| 确保最低收入 | 如果金价大涨，机会成本巨大 |

### 2. 看跌期权 Put Options

> **买保险**: 花一笔保费，获得"在某个价格以上卖出"的权利。如果金价涨了，你不用行使这个权利，直接按市场价卖更好。

```
例子:
- 当前金价: $4,000/oz
- 买一个 $3,800/oz 的看跌期权 (Put)
- 支付保费: $80/oz

如果到期时:
- 金价涨到 $5,000 → 不行使期权，按 $5,000 卖出 (实际收入: $5,000 - $80 = $4,920)
- 金价跌到 $3,000 → 行使期权，按 $3,800 卖出 (实际收入: $3,800 - $80 = $3,720)
```

| 优势 | 劣势 |
|---|---|
| **保留上涨收益** | 需要支付保费 (可能$50-150/oz) |
| 下跌有保护 | 保费是沉没成本 |
| 灵活 | 比远期成本高 |

### 3. 领子策略 Collar Strategy

> **零成本保险**: 同时买一个看跌期权(保护下跌)和卖一个看涨期权(放弃部分上涨)，两个期权的保费互相抵消，成本接近零。

```
例子:
- 当前金价: $4,000/oz
- 买 $3,600/oz 看跌 (下限保护)
- 卖 $4,800/oz 看涨 (上限让利)
- 净成本: ~$0 (两个保费互抵)

如果到期时:
- 金价涨到 $6,000 → 被迫在 $4,800 卖出 (上限限制)
- 金价维持 $4,000 → 按市价卖 $4,000 (在区间内)
- 金价跌到 $3,000 → 按 $3,600 卖出 (下限保护)

你的收入永远在 $3,600-$4,800 之间 — 有底有顶
```

| 优势 | 劣势 |
|---|---|
| 零/低成本 | 上涨收益被封顶 |
| 下跌有保护 | 结构较复杂 |
| 银行最喜欢 | 需要专业顾问帮助设计 |

## 对冲比例 How Much to Hedge?

> **行业惯例**: 矿企通常对冲 **15-60%** 的产量，而非100%。

| 场景 | 建议对冲比例 | 原因 |
|---|---|---|
| 新矿山建设期 | 50-80% | 确保还贷能力 |
| 稳定运营期 | 15-30% | 保留上涨弹性 |
| 金价高位 | 30-50% | 锁定利润 |
| 金价低位 | 10-20% | 保留反弹空间 |
| 面临收购 | 不对冲 | 保留标的价值 |

## 历史教训 Historical Lessons

### Barrick Gold 的对冲噩梦 (1999-2009)

> **最著名的对冲失败案例**: 1990年代末，Barrick Gold建立了巨大的远期对冲头寸（超过$800/oz锁定了数百万盎司的产量）。当金价在2000年代飙升至$1,000+/oz时，这些对冲头寸造成了**超过$100亿的机会损失**。公司被迫花了近10年时间才关闭所有对冲头寸。
>
> **教训**: 过度对冲等于投机。对冲是风险管理工具，不是赚钱手段。

### AngloGold Ashanti (类似教训)
- 也因过度对冲在金价上涨期间损失惨重
- CEO曾承认策略是"鲁莽的"

## PMH对冲建议

1. **勘探阶段**: 不需要对冲（还没有产量）
2. **开发阶段**: 如果需要银行贷款，准备好对冲方案（银行会要求）
3. **生产阶段**: 建议对冲 **不超过30%** 的年产量
4. **工具选择**: Collar最适合初创矿企（零成本+有保护）
5. **聘请顾问**: 对冲策略设计需要专业的商品交易顾问

---

*关键记忆: Forward=锁价(简单但丢掉上涨), Put=买保险(贵但保留上涨), Collar=零成本区间(最实用)。对冲15-60%产量，永远不要100%对冲。*
