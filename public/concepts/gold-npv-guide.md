# CEO的NPV速查指南 CEO's Quick NPV Guide

*参考资料: [Investopedia - NPV](https://www.investopedia.com/terms/n/npv.asp), [Corporate Finance Institute](https://corporatefinanceinstitute.com/resources/valuation/net-present-value-npv/)*

## 30秒理解NPV

> **一句话**: NPV告诉你"这个项目现在值多少钱"。如果NPV > 0，就值得投资。

> **生活类比**: 有人问你"投$100万，5年内每年收回$30万，值不值？"直觉上你会算5×30=$150万>$100万，好像赚了$50万。但NPV说"等等，未来的钱没有现在的钱值钱"——考虑通胀和风险后，这$150万放到今天其实只值$120万左右。所以你真正赚的是$120万-$100万=$20万（这就是NPV）。

## CEO需要记住的5个数字

| 指标 | 含义 | 投资决策 |
|---|---|---|
| **NPV > 0** | 项目创造价值 | 值得投资 |
| **NPV = 0** | 刚好回本 | 边缘项目，需谨慎 |
| **NPV < 0** | 项目毁损价值 | 不要投资 |
| **IRR > 25%** | 资本回报率优秀 | 西非金矿项目的最低门槛 |
| **回收期 < 3年** | 快速回本 | 降低政治风险窗口 |

## 一个真实的金矿NPV计算

假设PMH在马里发现一个金矿:

**基本假设:**
| 参数 | 数值 |
|---|---|
| 初始投资 (CAPEX) | $80M |
| 矿山寿命 | 10年 |
| 年产量 | 80,000 oz |
| 金价 | $2,500/oz |
| AISC | $1,200/oz |
| 权利金 + 税 | 收入的15% |
| 折现率 | 10% |

**计算:**
```
年收入 = 80,000 oz × $2,500/oz = $200M
年AISC = 80,000 oz × $1,200/oz = $96M
年税费 = $200M × 15% = $30M
年净现金流 = $200M - $96M - $30M = $74M

NPV = $74M × 6.14 (10年10%年金因子) - $80M
NPV = $454M - $80M = $374M ✅ 非常好的项目！

IRR ≈ 90%+ ✅ 远超25%门槛
回收期 ≈ $80M ÷ $74M = 1.1年 ✅ 远低于3年
```

## 金价敏感性 — CEO最该关注的一张表

| 金价 | 年净现金流 | NPV (10%) | IRR | 还值得投吗？ |
|---|---|---|---|---|
| $1,500 | $14M | +$6M | 12% | 勉强 |
| $2,000 | $44M | +$190M | 52% | 值得 |
| **$2,500** | **$74M** | **+$374M** | **90%+** | **非常值得** |
| $3,000 | $104M | +$559M | 130%+ | 极好 |
| $4,000 | $164M | +$927M | 200%+ | 梦幻级 |

> **核心洞察**: 金价每涨$500/oz，NPV增加约$185M。在当前$4,000+/oz的金价下，几乎任何品位合理的金矿项目NPV都非常高。**但CEO必须问：如果金价跌回$1,500，我的项目还活着吗？**

## CEO在看可行性报告时应该问的5个问题

1. **"你用了什么金价假设？"** — 如果用$4,000算，当然好看。用$1,800算还赚钱吗？
2. **"CAPEX有没有留余量？"** — 全球矿山建设平均超支30%。$80M很可能变成$100M+
3. **"折现率用了多少？"** — 8%和12%的NPV差异可以是翻倍。西非应该用10-12%
4. **"回收期多长？"** — 在马里，3年以内的回收期才安全（政治风险窗口）
5. **"IRR扣掉流送/权利金后还有多少？"** — 如果融资结构吃掉了大部分利润，IRR可能从90%降到30%
