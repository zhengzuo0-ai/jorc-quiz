# PRD: 黄金矿业完整知识库与学习平台

## Product Vision
将 JORC Quiz 从一个简单的题库工具，升级为**黄金矿业从业者的完整知识库和学习系统**，覆盖技术、商业、金融、人事和行业知识的全维度学习。

## Target User
- PMH (Paramount Mining Holdings) CEO 及团队
- 商业背景，矿业基础薄弱
- 主要关注黄金，之后扩展到铜
- 关注国家: 马里 (Mali)、老挝 (Laos)

## Core Principles
1. **学习效果至上** — 所有功能设计以"用户能学到并记住"为最终目标
2. **中英文对照** — 关键术语和名词必须双语
3. **实战导向** — 结合真实公司案例和行业实践
4. **循序渐进** — 从基础到高级，有清晰的学习路径
